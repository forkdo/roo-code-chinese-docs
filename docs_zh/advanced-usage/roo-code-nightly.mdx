# Roo Code Nightly

对于想要体验最新功能和更新的用户，Roo Code 提供了一个预发布版本，称为 **Roo Code Nightly**。每当代码合并到我们的主开发分支时，该版本会自动发布。这意味着它始终反映最新的代码状态。它专为希望访问最新功能并参与早期测试的用户而设计。

---

## 理解 Nightly 版本与其他预发布版本的区别

区分 Roo Code Nightly 与其他类型的预发布软件（如 beta 版本或发布候选版本）非常重要：

*   **Roo Code Nightly：**
    *   **频率：** 自动构建，每次合并到主开发分支时生成。
    *   **内容：** 反映最新的已合并代码。
    *   **稳定性：** 高度实验性；可能包含错误或未完成的功能。
    *   **测试：** 发布前几乎没有或没有手动测试。
    *   **受众：** 主要面向开发者、内部测试或愿意使用潜在不稳定软件的用户，他们希望看到绝对最新的变化，并帮助及早发现问题。

*   **一般预发布版本（如 Beta、发布候选版本）：**
    *   **频率：** 按计划定期发布（例如每周、每月），在达到特定里程碑时进行。
    *   **内容：** 代表更精心策划和规划的中间版本。
    *   **稳定性：** 比 Nightly 构建更稳定；通常经过一些手动测试或验证。
    *   **测试：** 通常会接受更集中的测试。
    *   **受众：** 面向更广泛的早期采用者或 beta 测试者，以在稳定版本发布前收集反馈。

简而言之：**Nightly = 最新代码，可能不稳定，更新频繁。** 其他预发布版本通常更新较少，经过更多验证，旨在进行更广泛的测试。

## 安装 Roo Code Nightly

要安装 Roo Code Nightly：

1.  打开 VS Code。
2.  访问扩展：点击活动栏中的扩展图标，或按 `Ctrl+Shift+X`（Windows/Linux）或 `Cmd+Shift+X`（macOS）。
3.  搜索 "Roo Code Nightly"。
4.  选择 Roo Code 发布的 "Roo Code Nightly"，点击 **安装**。
5.  如果提示，请重新加载 VS Code。
