# Roo Code Nightly

对于希望体验最新功能和更新的用户，Roo Code 提供了一个名为 **Roo Code Nightly** 的预发布版本，该版本会在每次代码变更合并到主开发分支时自动发布。这意味着它始终反映最新的代码状态。此版本专为希望使用最新功能并参与早期测试的用户设计。

---

## 了解 Nightly 版本与其他预发布版本的区别

区分 Roo Code Nightly 与其他类型的预发布软件（如 Beta 版本或候选发布版本）非常重要：

*   **Roo Code Nightly：**
    *   **发布频率：** 自动化构建，每次合并到主开发分支时生成。
    *   **内容：** 反映最新合并的代码。
    *   **稳定性：** 高度实验性；可能包含错误或未完成的功能。
    *   **测试：** 发布前极少或没有手动测试。
    *   **受众：** 主要面向开发人员、内部测试人员，或愿意使用可能不稳定的软件以查看最新变更并帮助早期发现问题的高级用户。

*   **通用预发布版本（例如 Beta、候选发布版本）：**
    *   **发布频率：** 定期发布（例如每周、每月），在达到特定里程碑时发布。
    *   **内容：** 代表更精心策划和计划的中间版本。
    *   **稳定性：** 比 Nightly 版本更稳定；通常经过一些手动测试或验证。
    *   **测试：** 通常接受更有针对性的测试。
    *   **受众：** 面向更广泛的早期采用者或 Beta 测试人员，以便在稳定版本发布前收集反馈。

简而言之：**Nightly = 最新代码，可能不稳定，频繁更新。** 其他预发布版本通常发布频率较低，经过更多验证，旨在进行更广泛的测试。

---

## 安装 Roo Code Nightly

要安装 Roo Code Nightly：

1.  打开 VS Code。
2.  访问扩展：点击活动栏中的扩展图标，或按 `Ctrl+Shift+X`（Windows/Linux）或 `Cmd+Shift+X`（macOS）。
3.  搜索 "Roo Code Nightly"。
4.  选择 Roo Code 发布的 "Roo Code Nightly" 并点击 **安装**。
5.  如果提示，请重新加载 VS Code。
