# Roo Code 2.1.14 版本说明

此版本包含对差异编辑的修复和改进。

---

## 改进与修复

*   修复了差异无法正确应用的问题。
*   尝试使用 Aider 的统一差异提示，以期获得更好的结果。
*   增加了逻辑，当启用差异编辑时，自动拒绝导致输出截断的 `write_to_file` 命令。
