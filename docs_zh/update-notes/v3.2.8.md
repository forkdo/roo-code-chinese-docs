# Roo Code 3.2.8 版本说明

此补丁版本包含对设置和提供商集成的修复。

---

## Bug 修复

*   修复了打开自定义模式设置 JSON 文件的问题。
*   将提供商密钥输入验证恢复为检查 `onInput` 而不是 `onChange`，以解决输入 API 密钥时的问题。（感谢 samhvw8！）
*   为 OpenAI 兼容提供商添加了使用 Azure 的显式复选框。（感谢 samhvw8！）
*   修复了 Glama 使用情况报告。（感谢 punkpeye！）
*   向 AWS Bedrock 提供商选项添加了 Llama 3.3 70B Instruct 模型。（感谢 Premshay！）
