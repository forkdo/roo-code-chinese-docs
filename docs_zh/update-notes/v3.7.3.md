# Roo Code 3.7.3 版本说明（2025-02-25）

此版本引入了通过 Anthropic API 对 Claude Sonnet 3.7 扩展“思考”功能的支持。

---

## 功能亮点

*   **扩展“思考”支持：** 通过 Anthropic 提供商增加了对 Claude Sonnet 3.7 扩展[“思考”](https://docs.anthropic.com/en/docs/build-with-claude/extended-thinking)的支持。这允许模型为复杂任务的内部推理分配更多 token，从而可能提高响应质量。（感谢 cte！）
