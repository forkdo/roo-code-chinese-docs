# Roo Code 3.18.4 版本说明 (2025-05-25)

此版本提升了代码库索引的可靠性，修复了若干用户界面问题，并更新了 AI 模型支持，增强了调试功能。

---

## Bug 修复

*   **代码库索引**：修复了设置保存问题，并改进了 Ollama 索引性能，使语义搜索更加可靠（感谢 daniel-lxs！）
*   **文件处理**：修复了当用户拒绝 apply_diff 操作时对字节顺序标记 (BOM) 的处理（感谢 avtc！）
*   **自动批准输入**：修复了启用自动批准时聊天输入框会错误清空的问题（感谢 Ruakij！）
*   **构建过程**：修复了 bootstrap 脚本参数，以便在开发设置期间更好地兼容 pnpm（感谢 ChuKhaLi！）

---

## 提供商更新

*   **xAI 模型**：更新了 xAI 模型列表和默认模型配置，以提升性能（感谢 PeterDaveHello！）

---

## 其他改进

*   **增强调试功能**：为所有提供商的 API 请求添加了元数据，以便更好地进行故障排除和监控（感谢 dtrugman！）
