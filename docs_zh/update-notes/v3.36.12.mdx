# Roo Code 3.36.12 Release Notes (2025-12-18)

此版本改进了 Gemini 和 OpenAI 模型的默认工具偏好设置，修复了 Bedrock 嵌入器识别问题，并增强了错误跟踪。

<img src="/img/v3.36.12/v3.36.12.png" alt="Roo Code v3.36.12 Release" width="600" />

## QOL 改进

* **为 Gemini 和 OpenAI 智能化工具默认设置**: Gemini 和 OpenAI 模型现在对文件编辑使用更好的默认工具，提高了开箱即用的可靠性。([#10170](https://github.com/RooCodeInc/Roo-Code/pull/10170))

## Bug 修复

* **Bedrock 嵌入器 CloudTrail 修复**: AWS Bedrock 用户在使用代码库索引时现在可以在 CloudTrail 日志中看到 Roo Code 的标识。(感谢 jackrein!) ([#10166](https://github.com/RooCodeInc/Roo-Code/pull/10166))

## 其他改进

* **更好的错误分组**: 改进了错误跟踪以加快问题解决速度。([#10163](https://github.com/RooCodeInc/Roo-Code/pull/10163))
