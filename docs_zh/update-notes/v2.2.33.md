# Roo Code 2.2.33 版本发布说明

此版本新增了“增强提示”功能，并改进了对 OpenAI 兼容提供商的支持。

---

## 功能亮点

*   添加了“增强提示”按钮（目前仅适用于 OpenRouter 模型）。
*   添加了对 OpenAI 兼容提供商列出可用模型的支持。（感谢 samhvw8！）
