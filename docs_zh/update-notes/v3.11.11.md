# Roo Code 3.11.11 版本说明 (2025-04-09)

此版本包含对 UI 交互的修复、改进的 OpenAI 兼容提供商选项、解析器更新以及其他增强功能。

---

## Bug 修复

*   修复与模式/配置文件下拉菜单的高亮显示交互问题 (感谢 atlasgong！)
*   修复终端工作目录逻辑 (感谢 KJ7LNW！)

---

## 改进

*   在 OpenAI 兼容提供商中添加设置 Host 标头和传统 OpenAI API 的功能，以更好地支持代理
*   改进 TypeScript、C++、Go、Java、Python 的 tree-sitter 解析器 (感谢 KJ7LNW！)
*   改进 `readFileTool` XML 输出格式 (感谢 KJ7LNW！)
*   支持符号链接的规则文件/目录，以实现更灵活的规则设置
*   通过 API 在侧边栏中运行任务时，将 Roo Code 聚焦到侧边栏
*   改进子任务 UI

---

## 提供商更新

*   添加 o1-pro 支持 (感谢 arthurauffray！)
