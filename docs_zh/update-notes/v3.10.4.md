# Roo Code 3.10.4 版本发布说明 (2025-03-25)

此版本带来了新的提供商支持、UI 增强以及多项改进和修复。

---

## 提供商与模型支持

*   为 Google Gemini 提供商添加了 Gemini 2.5 Pro 模型。(感谢 samsilveira！)
*   为 QWQ 模型在 OpenAI 兼容提供商设置中添加了 R1 支持复选框。(感谢 feifei325！)
*   为 `application-inference-profile` 添加了 Bedrock 支持。(感谢 maekawataiki！)

---

## UI/UX 改进

*   更新了聊天文本区域的 UX。(感谢 chadgauth！)
*   改进了 OpenRouter "过载" 错误消息的显示。

---

## 常规改进与修复

*   在命令面板中添加了"新任务"命令。(感谢 qdaxb！)
*   支持 TypeScript tree-sitter 查询中的测试声明。(感谢 KJ7LNW！)
*   支持读取 MCP 调用的图像响应。(感谢 nevermorec！)
*   支持任务的自定义存储路径。(感谢 Chenjiayuan195！)
*   动态获取创建/编辑自定义模式和 MCP 服务器的指令。(感谢 diarmidmackenzie！)
*   重命名并迁移了全局 MCP 和模式文件。(感谢 StevenTCramer！)
*   在 API 中添加了 `taskCreated` 事件，并提前订阅了 cline 事件。(感谢 wkordalski！)
*   为 McpHub 添加了 `watchPaths` 选项以检测文件更改。(感谢 01Rian！)
*   添加了控制工作区外读写自动批准的设置。
*   修复了 README 文档中的链接。(感谢 kvokka！)
*   修复了数字格式化后缀的国际化。(感谢 feifei325！)
*   修复了上下文提及建议中的打开标签支持。(感谢 aheizi！)
*   修复了系统提示预览中的浏览器工具可见性。(感谢 cannuri！)
*   修复了 OpenAI 模型的 `supportsPromptCache` 值。(感谢 PeterDaveHello！)
