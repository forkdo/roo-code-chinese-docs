# Roo Code 3.7.8 版本说明 (2025-02-27)

此版本新增对 GPT-4.5 Preview 的支持、Claude 优化功能，以及系统提示自定义高级功能。

---

## 提供商与模型支持

*   新增对 `gpt-4.5-preview` 的支持。
*   新增面向 Claude 模型的 Vertex AI 提示缓存支持。（感谢 aitoroses、lupuletic！）

---

## 高级功能

*   **自定义系统提示（仅限高级用户）：** 可通过在工作区创建 `.roo/system-prompt-[slug]` 文件，完全替换模式的系统提示。**警告：** 此操作会绕过内置安全保护机制，存在高度不可预测行为的风险。请谨慎使用。
