# Roo Code 3.3.4 版本说明

此版本带来了显著更快的差异编辑性能、可配置的 MCP 超时设置，以及增强的代码操作功能。

---

## 功能亮点

*   **更快的差异编辑：** 大幅提升了应用差异编辑的速度（最高可达 10 倍）。（感谢 hannesrudolph、KyleHerndon！）
*   **可配置的 MCP 超时：** 为 MCP 添加了每台服务器的网络超时配置，范围从 15 秒到 1 小时。
*   **增强的代码操作：** 在右键菜单、问题选项卡和灯泡指示器中添加了“解释/改进/修复”代码操作。添加了选项，可在当前任务或新任务中执行这些操作。（感谢 samhvw8！）
