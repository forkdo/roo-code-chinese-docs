# Roo Code 3.26.0 版本发布说明 (2025-08-26)

此版本引入了 Grok Code Fast（前身为 Sonic）、内置 /init 命令以及对 Qwen Code CLI API 的支持。

<img src="/img/v3.26.0/v3.26.0.png" alt="Roo Code v3.26.0 Release" width="600" />

## Grok Code Fast

你可能已经猜到了，我们的隐身模型 Sonic 正式亮相了！ ([#7426](https://github.com/RooCodeInc/Roo-Code/pull/7426))

来自 xAI 的这款模型针对编码任务进行了优化，并因其以下特点而广受社区喜爱：

- **敏锐的推理**能力
- **大规模计划执行**
- **具有 UI 品味和直觉的代码建议**

如果你已经在 Roo Code Cloud 中体验过 Sonic，你将被自动切换到 Grok Code Fast。模型 `xai/grok-code-fast-1` 也可在 [xAI Provider](/providers/xai) 下使用，且并非免费（当免费期于 2025 年 8 月 28 日结束时）。

非常感谢我们的合作伙伴 xAI 以及所有用户——在隐身期间，超过 1000 亿个 token（并且还在增加！）通过 Sonic 运行！你们的惊人采用率和宝贵反馈将 Grok Code Fast 塑造成了今天如此强大的模型。

**重要提示：** 在促销期间，通过 [Roo Code Router](/providers/roo-code-router) 访问 Grok Code Fast 仍然是**免费的**。一旦定价确定，直接通过 xAI 提供商使用将产生标准费用。

> **📚 文档**：请参阅 [Roo Code Router](/providers/roo-code-router) 获取免费访问权限，或参阅 [xAI Provider](/providers/xai) 进行直接配置。

## 内置 /init 命令

我们新增了一个 /init 斜杠命令，用于项目入门 ([#7381](https://github.com/RooCodeInc/Roo-Code/pull/7381), [#7400](https://github.com/RooCodeInc/Roo-Code/pull/7400))：

- **自动项目分析**：分析整个代码库并创建全面的 AGENTS.md 文件
- **AI 助手优化**：生成文档，使 AI 助手能够立即在代码库中高效工作
- **模式特定指导**：为不同的 Roo Code 模式（代码、调试、架构师等）创建定制文档

/init 命令通过记录项目特定信息（这些信息无法仅从代码结构中明显看出），帮助 LLM 理解项目的独特模式和约定。

> 📚 文档：请参阅 [Slash Commands - The init command](/features/slash-commands#the-init-command) 了解详情。

## Qwen Code CLI API 支持

我们集成了 Qwen Code CLI 工具，允许 Roo Code 利用其免费访问层级来使用阿里巴巴的 Qwen3 Coder 模型 ([#7380](https://github.com/RooCodeInc/Roo-Code/pull/7380))：

- **免费推理**：通过 OAuth 利用 Qwen Code CLI 慷慨的免费层级（每天 2000 次请求，每分钟 60 次请求，无 token 限制），在促销期间可用。
- **100 万上下文窗口**：在单个对话中处理整个代码库。
- **无缝设置**：如果你已经对 Qwen Code CLI 工具进行了身份验证，则会自动工作。

此集成通过使用 Qwen Code CLI 的本地身份验证，为 Qwen3 Coder 模型提供免费访问。

> **📚 文档**：请参阅 [Qwen Code CLI Provider](/providers/qwen-code) 进行设置和配置。
