# Roo Code 2.2.0 版本说明

此版本引入了对模型上下文协议 (MCP) 的支持。

---

## 功能亮点

*   **模型上下文协议 (MCP) 支持：** 添加了 MCP 支持，使 Roo Code 能够使用自定义外部工具和服务（例如网络搜索、GitHub 工具）。
*   **MCP 服务器管理：** 添加了 MCP 服务器管理选项卡（可通过服务器图标访问），用于配置和管理连接。
*   **动态 MCP 服务器创建：** 添加了 Roo Code 根据用户请求动态创建新 MCP 服务器的功能。
