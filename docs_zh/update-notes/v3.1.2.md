# Roo Code 3.1.2 发布说明

此补丁版本添加了实验性的 Copilot 支持、模糊搜索改进和修复。

---

## 更新与修复

*   添加了对 VS Code 语言模型的实验性支持，包括 Copilot。（感谢 RaySinner, julesmons！）
*   修复了与配置文件切换相关的错误。（感谢 samhvw8！）
*   改进了 @-mentions、历史记录和模型列表中的模糊搜索。（感谢 samhvw8！）
*   为 Glama 提供商添加了 PKCE 支持。（感谢 punkpeye！）
*   为 `o1` 系统提示使用 'developer' 消息角色。
