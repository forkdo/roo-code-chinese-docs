# Roo Code 3.16.4 版本发布说明 (2025-05-09)

此版本包含针对提供商配置文件管理、终端焦点、自定义请求头保存、竞态条件、显示问题和错误处理等方面的改进和修复。

---

## 用户体验改进

*   改进外部 API 中的提供商配置文件管理
*   在预览中显示格式正确的多行命令（感谢 KJ7LNW！）
*   在 read_file 工具中优雅处理不支持的语言错误（感谢 KJ7LNW！）
*   增强 select-dropdown 的焦点样式并修复文档 URL（感谢 zhangtony239！）
*   正确处理 UI 中的模式名称溢出问题（感谢 elianiva！）

---

## 其他改进

*   正确保存 OpenAI 兼容的自定义请求头

---

## 提供商更新

*   通过使用 'only' 参数并禁用回退，强制在 OpenRouter 中选择提供商（感谢 shariqriazz！）

---

## Bug 修复

*   修复长配置文件名称的显示问题（感谢 cannuri！）
*   防止命令执行后粘贴时终端焦点被抢占（感谢 MuriloFP！）
*   修复更新提示时的竞态条件（感谢 elianiva！）
*   修复高对比度主题下的显示问题（感谢 zhangtony239！）
*   修复无法在 OpenRouter 上使用特定提供商的问题（感谢 daniel-lxs！）
*   修复项目 MCP 始终允许的问题（感谢 aheizi！）
