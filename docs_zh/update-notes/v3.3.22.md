# Roo Code 3.3.22 版本说明 (2025-02-20)

此版本包含对设置、MCP 服务器管理、语言支持和错误修复的常规改进。

---

## 常规改进

*   添加了删除 MCP 服务器的按钮。(感谢 hannesrudolph！)
*   为命令执行自动批准添加了通配符 (`*`) 选项（请谨慎使用！）。
*   增强了“提供商设置”UI，添加了清晰的“保存”按钮和关于未保存更改的警告。(感谢 System233！)
*   添加了在“提示”选项卡上设置自定义首选语言的支持。
*   添加了加泰罗尼亚语语言支持。(感谢 alarno！)

---

## 提供商支持

*   改进了对 Ollama 模型 `<think>` 推理标签的解析。(感谢 System233！)

---

## 错误修复

*   修复了系统提示预览复制按钮始终复制“代码”模式版本的问题。
*   修复了从“提示”选项卡添加自定义模式时 `.roomodes` 文件未被自动创建的问题。
