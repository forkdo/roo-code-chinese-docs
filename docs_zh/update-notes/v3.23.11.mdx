# Roo Code 3.23.11 版本说明 (2025-07-15)

此版本增加了新的提供商选项，并改进了模式导航。

## 提供商更新

* **Groq Kimi K2**：为 Groq 用户添加了 Kimi K2 模型选项，改进了具有相同最大令牌数和上下文窗口的模型的令牌处理可靠性

## 用户体验改进

* **模式导航**：添加了双向模式循环功能，使用 Cmd+Shift+. 快捷键可切换到上一个模式，当您越过目标模式时，可以更有效地进行模式导航（感谢 mkdir700！）([#5695](https://github.com/RooCodeInc/Roo-Code/pull/5695))

## 其他改进

* **GitHub Actions**：通过从 CI/CD 配置中移除无效参数，修复了 GitHub Actions 工作流警告（感谢 hamirmahal！）([#5676](https://github.com/RooCodeInc/Roo-Code/pull/5676))
