# Roo Code 3.11.10 版本发布说明 (2025-04-08)

此补丁版本包含对规则目录和提供商缓存的 bug 修复，以及对命令输出处理、翻译和代码清理的改进。

---

## Bug 修复

*   修复了嵌套 `.roo/rules` 目录无法被正确识别的问题（感谢 @taisukeoe！）。
*   修复了 OpenAI 兼容提供商的缓存使用跟踪问题。

---

## 常规和体验改进

*   更有效地处理聊天行中的长命令输出（感谢 @samhvw8！）。
*   为 zh-CN 添加了自定义翻译说明（感谢 @System233！）。
*   在将速率限制设置为每个配置文件后进行了代码清理（感谢 @ross！）。
