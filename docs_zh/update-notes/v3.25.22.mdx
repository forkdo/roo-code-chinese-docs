# Roo Code 3.25.22 版本说明 (2025-08-22)

此版本为 Groq 上的 Kimi K2 添加了提示缓存支持，并提升了全局自定义指令的可发现性。

## 功能

* **Groq 上的 Kimi K2 提示缓存**：为 Groq 上的 Kimi K2 模型添加了提示缓存支持，对缓存的输入 token 提供 50% 的折扣，以提高成本效率 ([#7324](https://github.com/RooCodeInc/Roo-Code/pull/7324))

## 体验优化

* **全局自定义指令文档**：在用户界面中添加了有用的文档链接，使全局自定义指令更易于发现和理解 ([#7114](https://github.com/RooCodeInc/Roo-Code/pull/7114))
