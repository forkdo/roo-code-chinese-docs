# Roo Code 3.11.14 版本发布说明 (2025-04-11)

此版本增强了规则文件处理并强制实施了文件读取设置。

---

## 改进

*   支持规则文件夹中的符号链接（指向目录和其他符号链接）（感谢 taisukeoe！）
*   更严格地强制执行始终完整读取文件而非部分读取的设置
