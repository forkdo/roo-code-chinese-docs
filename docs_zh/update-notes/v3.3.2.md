# Roo Code 3.3.2 版本说明

此版本改进了模式配置、OpenRouter 集成和 UI 元素。

---

## 改进与修复

*   在“提示词”标签页中添加了用于为特定模式选择 API 配置的下拉菜单。
*   修复了 MCP 工具的“始终允许”复选框不显示的问题。
*   改进了 OpenRouter DeepSeek-R1 集成：将温度设置为 0.6 并显示推理输出。（感谢 Szpadel！）
*   允许指定自定义 OpenRouter 基础 URL。（感谢 dairui1！）
*   改进了嵌套设置的 UI。（感谢 PretzelVector！）
