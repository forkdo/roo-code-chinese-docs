# Roo Code 3.7.12 版本说明 (2025-03-03)

此版本增强了上下文处理、思考能力和配置选项。

---

## 思考与上下文管理

*   将思考模型的最大 token 扩展至 128k，最大思考预算扩展至超过 100k。(感谢 monotykamary！)
*   在 Anthropic 提供商中使用 `count_tokens` API 以更准确地管理上下文窗口。
*   将 OpenRouter 的中间向外压缩默认设置为“开启”。
*   如果模式不支持 MCP，则从提示中排除 MCP 指令。

---

## 配置改进

*   添加了禁用浏览器工具的复选框。
*   如果检查点加载时间过长，则显示警告。
*   更新了 VS Code 语言模型 API 的警告文本。

---

## Bug 修复

*   修复了键盘模式切换器未更新 API 配置文件的问题。(感谢 aheizi！)
*   在欢迎屏幕上正确填充了默认的 OpenRouter 模型。
