# Roo Code 3.28.12 版本说明 (2025-09-29)

此版本修复了 Anthropic Sonnet 4.5 模型 ID 并添加了 Bedrock 1M 上下文复选框支持。

## Bug 修复

* **Anthropic Sonnet 4.5 模型 ID**：将模型 ID 修正为官方 `claude-sonnet-4-5`，解决了使用 Claude Sonnet 4.5 时的 API 错误 ([#8384](https://github.com/RooCodeInc/Roo-Code/pull/8384))

## 体验优化

* **Bedrock 1M 上下文复选框**：添加复选框以启用 Bedrock 上 Claude Sonnet 4 和 4.5 的 1M 上下文窗口支持，为大型项目提供更多灵活性 ([#8384](https://github.com/RooCodeInc/Roo-Code/pull/8384))
