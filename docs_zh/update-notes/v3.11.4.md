# Roo Code 3.11.4 版本说明 (2025-04-02)

此版本包含对 webview 状态的修复、Windows 单元测试的修复、tree-sitter 增强功能以及其他一些改进和错误修复。

---

## 错误修复

*   在当前任务被清除时，正确地向 webview 发送状态（感谢 wkordalski！）。
*   修复了单元测试，使其能在 Windows 上正常运行（感谢 StevenTCramer！）。
*   修复了 `apply_diff` 中删除操作的行号剥离问题。
*   修复了从下拉菜单访问时提示和设置标签页无法滚动的问题。
*   修复了 Bedrock 的提示增强功能（感谢 Smartsheet-JB-Brown！）。
*   修复了 tree-sitter 行号的差一错误。

---

## 用户体验改进

*   Tree-sitter 增强功能：支持 TSX、TypeScript、JSON 和 Markdown（感谢 KJ7LNW！）。
*   更新了历史记录选择模式按钮的间距（感谢 kyle-apex！）。
*   将下拉菜单的高度限制为视口的 80%（感谢 axmo！）。
*   在 API 失败时启用模型选择（感谢 kyle-apex！）。
*   将 AWS 区域下拉菜单更新为最新数据（感谢 Smartsheet-JB-Brown！）。
*   使扩展图标在不同主题下更易于识别。

---

## 其他改进

*   通过 `npm audit fix` 更新依赖项（感谢 PeterDaveHello！）。
*   允许进程通过 Unix 套接字访问 Roo Code API。
*   改进了 zh-TW 繁体中文翻译（感谢 PeterDaveHello！）。
*   添加了对 Azure AI 推理服务与 DeepSeek-V3 模型的支持（感谢 thomasjeung！）。
*   移除了实验性的统一差异功能。
