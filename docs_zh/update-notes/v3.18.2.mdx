# Roo Code 3.18.2 发布说明（2025-05-23）

本次发布引入了上下文压缩功能的增强以及多项重要错误修复，以改进设置管理和终端集成。

---

## 上下文压缩功能增强

通过新增设置和改进功能，增强了上下文压缩特性：

*   **高级控制**：新增实验性设置，用于精细调整对话压缩行为
*   **改进的压缩效果**：在保留重要上下文的同时，实现更优的对话摘要
*   **UI 增强**：新增用于管理压缩设置的界面组件

(thanks SannidhyaSah!)

---

## 错误修复

*   **全局设置导出**：修复了全局设置导出功能，确保配置可在不同环境间正确传输
*   **文件选择器图标**：修复了文件选择器中 vscode-material-icons 的显示问题，提升视觉一致性
*   **终端集成**：修复终端集成问题，使其正确遵循用户配置的超时设置（感谢 KJ7LNW!）
