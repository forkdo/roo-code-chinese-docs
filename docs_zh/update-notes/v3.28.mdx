# Roo Code 3.28 综合版本说明

此版本为 Roo Code Cloud 引入了 Supernova 模型，Task Sync 和 Roomote Control 功能可让您通过手机监控和控制任务，还包含 Roo Code Cloud 集成功能、增强的聊天编辑、对话保存改进以及多项错误修复。

## Supernova 模型 - 免费使用

我们已将全新的 **roo/code-supernova** 隐身模型添加到 Roo Code Cloud 提供商 - **促销期间完全免费** ([#8175](https://github.com/RooCodeInc/Roo-Code/pull/8175))：

- **免费使用**：无需 API 密钥，无需付费 - 通过 Roo Code Cloud 完全免费
- **高级功能**：强大的 AI 模型，支持**图像功能**，适用于多模态编码任务
- **上下文窗口适中**：200,000 个 token 的上下文窗口，最大输出 16,384 个 token
- **隐身性能**：体验 code-supernova 变体的尖端模型性能
- **零配置**：只需从 Roo Code Cloud 提供商中选择即可 - 无需任何配置

从 Roo 提供商中选择**免费**的 code-supernova 模型，即可**零成本**开始使用这一强大新功能。

> **文档**：查看 [Roo Code Router 指南](/providers/roo-code-router) 了解模型选择详情。

## Task Sync 和 Roomote Control

推出我们的新云连接功能，让您能够**通过手机监控和控制长时间运行的任务** - 不再需要守在办公桌前！ ([#7799](https://github.com/RooCodeInc/Roo-Code/pull/7799), [#7805](https://github.com/RooCodeInc/Roo-Code/pull/7805), [#7850](https://github.com/RooCodeInc/Roo-Code/pull/7850))：

> **重要提示**：Roo Code 仍然**完全免费且开源**。Task Sync 和 Roomote Control 是可选的补充服务，可将您的本地 VS Code 连接到云端 - 所有处理仍在您的 VS Code 实例中进行。

<img src="/img/v3.28.0/v3.28.0.png" alt="Roomote Control 界面，显示从浏览器实时监控和控制任务" width="300" />

**[Task Sync](/roo-code-cloud/task-sync) (所有用户免费)**：
- **随时随地监控**：离开办公桌时可通过手机查看长时间运行的任务
- **实时更新**：本地任务消息和进度的实时流式传输
- **任务历史**：您的任务会保存到云端，方便日后参考
- **云端可见性**：可通过任何浏览器查看您的 VS Code 任务

**[Roomote Control](/roo-code-cloud/roomote-control)**：
- **远程继续任务**：通过手机继续任务 - 响应提示、修复错误、批准操作
- **完整聊天控制**：如同在 IDE 中一样与聊天框交互
- **启动/停止任务**：随时随地启动新任务或终止正在运行的任务
- **完全控制**：从任何地方对本地 VS Code 进行双向完全控制

Task Sync 让您能够从任何设备监控本地开发环境。添加 Roomote Control 可获得完整的远程控制功能 - 无论您使用的是另一台电脑、平板电脑还是智能手机。

> **文档**：查看 [Task Sync](/roo-code-cloud/task-sync) 和 [Roomote Control 指南](/roo-code-cloud/roomote-control)。
