# Roo Code 3.0 版本发布说明 (2025-02-27)

此发布周期标志着 Roo Code 中聊天模式的引入，允许用户与 AI 进行架构讨论和代码库问题交流，而无需立即进入代码生成。此外，还包含了一些小的修复和更新。

---

## 主要功能：聊天模式 (v3.0.0)

*   **聊天模式介绍：** Roo Code 现在具有不同的聊天模式（Ask、Architect、Code）。这允许用户：
    *   询问有关系统架构或代码库的问题。
    *   在代码实现之前进行更高层次的讨论。
    *   为每种模式分配不同的 API 配置配置文件，从而实现使用不同的模型进行思考和编码。
*   **Roo Code Reddit 社区：** 推出了 [Roo Code Reddit](https://www.reddit.com/r/RooCode)，用于反馈和社区互动。

---

## Bug 修复和次要更新

*   **Reddit 链接修复 (v3.0.1)：** 更正了应用程序中的 Reddit 链接。
*   **聊天输入视觉故障 (v3.0.1)：** 修复了聊天输入中的一个小视觉故障。
*   **按钮对齐调整 (v3.0.2)：** 对聊天输入中的按钮对齐进行了细微调整。
*   **VSCode 引擎要求 (v3.0.3)：** 将所需的 VSCode 引擎更新为 `^1.84.0`，以与 Cline 保持一致。
