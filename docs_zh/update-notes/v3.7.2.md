# Roo Code 3.7.2 版本发布说明 (2025-02-24)

此补丁版本包含与 Claude Sonnet 3.7 集成相关的修复以及提示调整。

---

## Bug 修复与改进

*   修复了 OpenRouter 的 `anthropic/claude-3.7-sonnet:beta` 的 computer use 和提示缓存功能。（感谢 cte！）
*   修复了 Sonnet 3.7 的滑动窗口计算问题，该问题曾导致上下文窗口溢出。（感谢 cte！）
*   在系统提示中更强烈地鼓励使用差异编辑。（感谢 hannesrudolph！）
