# Roo Code 3.35.3 发布说明（2025-12-02）

本次发布通过简化欢迎界面提升了用户上手体验，并为 API 提供方新增了对隐身模型的支持。

## 质量与体验改进

* **全新欢迎界面**：采用整合组件的简化欢迎界面，提供更简洁、一致的上手体验（[#9741](https://github.com/RooCodeInc/Roo-Code/pull/9741)）

## 其他改进

* **隐身模型隐私**：在 Roo API 中标记为 "stealth" 的模型，其系统提示词中将包含供应商保密指令，从而支持白标或匿名模型体验（[#9742](https://github.com/RooCodeInc/Roo-Code/pull/9742)）
