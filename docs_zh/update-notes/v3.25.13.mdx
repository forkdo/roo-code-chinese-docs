# Roo Code 3.25.13 发布说明 (2025-08-13)

此版本为 AWS Bedrock 添加 Sonnet 1M 上下文支持，改进国际化，并修复搜索功能。

## 功能

* **Sonnet 1M 上下文支持**: 在使用 AWS Bedrock 时为 Claude Sonnet 4 启用 1M token 上下文窗口，允许您处理更大的文档并进行更长的对话 ([#7032](https://github.com/RooCodeInc/Roo-Code/pull/7032))

## 生活质量改进

* **国际化**: 账户按钮现在以您选择的语言显示，而不是始终显示英文（感谢 zhangtony239！）([#6978](https://github.com/RooCodeInc/Roo-Code/pull/6978))
* **Agent 文件灵活性**: Roo Code 现在识别 AGENT.md 和 AGENTS.md 文件用于 agent 规则，优先检查 AGENTS.md 以保持兼容性（感谢 Brendan-Z！）([#6913](https://github.com/RooCodeInc/Roo-Code/pull/6913))

## 错误修复

* **搜索工具可靠性**: 修复了 search_files 工具在遇到权限被拒绝的文件时失败的问题 - 现在它会继续工作并返回可访问文件的结果（感谢 R-omk！）([#6757](https://github.com/RooCodeInc/Roo-Code/pull/6757))

## 提供商更新

* **OpenAI**: 从可用选项中移除了已弃用的 GPT-4.5 Preview 模型，因为它已从 OpenAI API 中移除（感谢 PeterDaveHello！）([#6948](https://github.com/RooCodeInc/Roo-Code/pull/6948))
