# Roo Code 3.16.2 发布说明 (2025-05-07)

此版本包含更清晰的 XML tool 使用格式化说明，便于理解，并改进了错误处理，提供更稳定的体验。

---

## Tool Use 改进
*   **澄清 XML Tool 格式化说明**：文档和提示现在提供了更清晰的 XML tool 使用格式示例，防止 `<tool_name>` 和其他 tool use 错误。
*   此修复主要针对使用工具时 Gemini 2.5 遇到的问题

---

## 其他改进
*   **改进流式传输的错误处理**：修复了应用程序可能卡住等待响应的问题。应用程序现在能够在流式传输期间优雅地从错误中恢复，减少无响应行为的可能性，提高可靠性。（感谢 monkeyDluffy6017！）
