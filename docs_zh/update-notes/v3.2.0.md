# Roo Code 3.2.0 版本说明

此版本标志着从 Roo Cline 到 Roo Code 的品牌重塑，并引入了自定义模式功能！

---

## 功能亮点

*   **名称变更：** 该扩展已从 Roo Cline 更名为 Roo Code。
*   **自定义模式：** 您现在可以创建自己的自定义模式（角色），超越内置的 Code（代码）、Architect（架构师）和 Ask（问答）模式。通过定义自定义提示词并选择每个模式可访问的工具，创建专门的助手。只需输入“Create a new mode for...”（为...创建新模式）或访问 Prompts（提示词）选项卡即可开始。
