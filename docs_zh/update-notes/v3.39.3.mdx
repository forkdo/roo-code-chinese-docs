# Roo Code 3.39.3 版本说明 (2026-01-10)

此版本将 Roo Code Cloud Provider 重命名为 Roo Code Router，并改进了 CLI 和内部组织结构。

<img src="/img/v3.39.3/v3.39.3.png" alt="Roo Code v3.39.3 Release" width="600" />

## Roo Code Router 重命名

Roo Code 现在将扩展中所有提及 **Roo Code Cloud Provider** 的地方都显示为 **Roo Code Router**。新名称旨在更清晰地表达其意图：Roo Code Router 是一种路由器式提供商（类似于其他路由器/聚合器服务的理念），但提供的是**我们已在 Roo Code 中测试过的精选模型集**。 ([#10560](https://github.com/RooCodeInc/Roo-Code/pull/10560))。

## 生活质量改进

* **Roo Code Router 命名一致性**：更新 Roo Code Router 服务名称和链接，使 UI 指向正确的目标 ([#10607](https://github.com/RooCodeInc/Roo-Code/pull/10607))
* **Roo Code Router 命名一致性**：更新共享类型中的路由器命名，使标签在整个 UI 中保持一致 ([#10605](https://github.com/RooCodeInc/Roo-Code/pull/10605))
* **ExtensionHost 组织**：改进模式更改及相关事件的内部组织结构，以帮助保持行为一致性 ([#10600](https://github.com/RooCodeInc/Roo-Code/pull/10600))
* **CLI 版本测试**：添加选项以在本地构建和安装 CLI 版本进行测试 ([#10597](https://github.com/RooCodeInc/Roo-Code/pull/10597))
* **CLI 组织**：重组 CLI 文件结构，使未来的更改更易于维护 ([#10599](https://github.com/RooCodeInc/Roo-Code/pull/10599))

## 其他改进

* **CLI TUI**：为 Roo CLI 添加交互式终端 UI 模式 ([#10480](https://github.com/RooCodeInc/Roo-Code/pull/10480))
