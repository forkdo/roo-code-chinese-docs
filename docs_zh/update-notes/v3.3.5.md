# Roo Code 3.3.5 版本发布说明

此版本带来了上下文窗口可见性、模式切换的自动批准、新的实验性编辑工具以及对 DeepSeek 的改进。

---

## 功能亮点

*   **上下文窗口可见性：** 在任务标题中显示对话的令牌计数和上下文容量信息，并使其对模型可见。（感谢 MuriloFP！）
*   **自动批准模式切换：** 添加了复选框以自动批准模式切换请求。（感谢 MuriloFP！）
*   **实验性编辑工具：** 新增实验性工具：`insert_content`（在指定行号插入文本）和 `search_and_replace`（替换文本/正则表达式）。（感谢 samhvw8！）

---

## 提供商改进

*   **DeepSeek R1：** 通过捕获推理过程、支持更多 OpenRouter 变体、修复空数据块导致的崩溃以及避免系统消息，改进了支持。（感谢 Szpadel！）
