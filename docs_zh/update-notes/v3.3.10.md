# Roo Code 3.3.10 发布说明

本次发布包含对模式切换和提示词的显著改进、实验性差异功能的增强，以及各种通用修复。

---

## 显著变更

*   改进了 Architect 和 Ask 模式的默认提示词。
*   支持通过斜杠命令在模式之间切换（例如：`/ask why is the sky blue?`）。

---

## 实验性功能

*   改进了代码操作中的实验性统一差异策略和选择逻辑。（感谢 nissa-seru！）

---

## 通用改进与修复

*   在 `@`-提及文件建议中添加了当前打开标签页的快捷方式。（感谢 olup！）
*   启用了 `o3` 和 `o1` 模型响应中的 Markdown 格式。（感谢 nissa-seru！）
*   改进了终端 shell 检测逻辑。（感谢 canvrno、nissa-seru！）
*   对“提示词”标签页中的模式列表进行了视觉优化和清理。
*   修复了 `o1-mini` 的定价问题。（感谢 hesara！）
*   修复了上下文窗口大小的计算问题。（感谢 MuriloFP！）
*   修复了在切换 API 配置文件时偶尔出现的错误。（感谢 samhwv8！）
*   修复了提供商下拉菜单中的双滚动条问题。
