# Roo Code v3.25.20 发布说明 (2025-08-20)

此版本为 Roo Code 用户引入了 Sonic 模型，并提供了公告帮助用户发现新模型。

## QOL 改进

* **Sonic 模型公告**: 添加了公告以通知用户 Sonic 模型的可用性，使发现新模型选项变得更加容易 (感谢 mrubens!) ([#7244](https://github.com/RooCodeInc/Roo-Code/pull/7244))
* **发布管理**: 为 v3.25.20 添加了 changeset 以确保正确的版本管理 (感谢 mrubens!) ([#7245](https://github.com/RooCodeInc/Roo-Code/pull/7245))

## Provider 更新

* **Sonic 模型支持**: 在 provider 系统中添加了对 Sonic 模型的支持 (感谢 mrubens!) ([#7246](https://github.com/RooCodeInc/Roo-Code/pull/7246))
