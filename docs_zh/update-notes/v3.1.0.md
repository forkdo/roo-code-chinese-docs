# Roo Code 3.1.0 发布说明

本次发布新增了对聊天模式提示和指令的自定义功能，并改进了“Enhance Prompt”功能。

---

## 功能亮点

*   **可自定义的模式提示**：可通过“Prompts”标签页或模式特定的 `.clinerules-mode` 文件，为每个聊天模式（Code、Architect、Ask）自定义角色定义和指令。
*   **重构的 Enhance Prompt**：“Enhance Prompt”按钮现已支持任何提供者和 API 配置，可使用完全自定义的提示来构建消息。
*   新增了用于复制聊天消息中 Markdown 内容的按钮。
