# Roo Code 3.11.15 版本说明 (2025-04-13)

此版本增加了任务历史记录过滤、本地化、UI 选项，并包含多项服务提供商更新和错误修复。

---

## 任务历史记录过滤

*   增加了按工作区过滤任务历史记录的功能。(感谢 samhvw8！)
*   默认情况下，仅显示当前工作区的任务。
*   在历史记录视图中勾选 `显示所有工作区的任务` 选项可查看全局任务历史记录。

---

## 改进

*   改进了添加新设置的文档 (感谢 KJ7LNW！)
*   本地化 package.json (感谢 samhvw8！)
*   增加隐藏欢迎消息的选项，并修复新配置文件对话框的背景颜色 (感谢 zhangtony239！)
*   恢复 VSCodeButton 组件的焦点环 (感谢 pokutuna！)

---

## 错误修复

*   修复 .tool-versions 文件中的 Node.js 版本 (感谢 bogdan0083！)
*   修复打开标签页的重复建议提及 (感谢 samhvw8！)
*   修复使用配置文件时的 Bedrock ARN 验证和令牌过期问题 (感谢 vagadiya！)

---

## 服务提供商更新

*   增加 Anthropic 选项，允许将 API 令牌作为 Authorization 标头传递，而不是 X-Api-Key (感谢 mecab！)
