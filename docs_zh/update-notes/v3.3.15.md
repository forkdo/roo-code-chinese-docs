# Roo Code 3.3.15 版本发布说明 (2025-02-08)

此版本将检查点作为可选实验性功能引入，并包含用户体验改进和错误修复。

---

## 功能亮点

*   **检查点（实验性可选功能）：** 在高级设置中作为可选功能引入了检查点。这允许在任务期间跟踪项目变更，以便审查或回滚。在此实验阶段需要用户反馈。

---

## 用户体验改进

*   为最近任务列表添加了复制按钮。（感谢 hannesrudolph！）
*   增强了添加新 API 配置文件的流程。

---

## 错误修复

*   解决了设置界面上 API 配置文件切换的问题。
*   改进了 MCP 初始化和服务器重启。（感谢 MuriloFP、hannesrudolph！）
