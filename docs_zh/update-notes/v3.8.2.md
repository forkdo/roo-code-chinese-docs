# Roo Code 3.8.2 发布说明 (2025-03-08)

本次发布新增了子任务的自动批准功能、UI 改进和错误修复。

---

## 功能亮点

*   为 Boomerang 任务（子任务）的创建和完成添加了自动批准切换功能。（感谢 shaybc！）
*   在使用多 diff 编辑策略时添加了进度指示器。（感谢 qdaxb！）
*   为 OpenAI 兼容提供者添加了 `o3-mini` 支持。（感谢 yt3trees！）

---

## 错误修复

*   修复了导致文件开头出现乱码字符的编码问题。
*   修复了在某些情况下设置下拉菜单被截断的问题。
