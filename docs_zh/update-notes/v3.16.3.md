# Roo Code 3.16.3 版本说明 (2025-05-08)

此版本（暂时）回退了 Tailwind CSS 迁移以恢复 UI 稳定性，并为语言解析器添加了 Elixir 文件扩展名支持，以增强代码分析功能。

---

## 其他改进
*   **回退 Tailwind 迁移**：通过回退 Tailwind CSS 迁移恢复了之前的用户界面。这将 UI 恢复到熟悉且稳定的状态，解决了迁移引入的问题，确保用户看到预期的界面，而不会出现意外的布局或样式更改。
*   **在语言解析器中添加 Elixir 文件支持**：在语言解析器中添加了对 Elixir (`.ex`, `.exs`) 文件的支持。这扩展了语言支持，允许用户无缝处理 Elixir 代码，并实现更好的代码分析，从而提高工作效率。（感谢 pfitz！）
