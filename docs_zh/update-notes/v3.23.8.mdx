# Roo Code 3.23.8 版本说明 (2025-07-13)

此版本增加了命令工作流控制功能，并改进了用户界面。

## 命令拒绝列表

我们添加了在工作流中自动拒绝不需要的命令的功能（感谢 hannesrudolph！）：

- **始终拒绝**：将命令标记为“始终拒绝”，以防止意外执行
- **节省时间**：无需重复手动拒绝相同的命令
- **工作流控制**：通过“始终拒绝”选项补充现有的自动批准功能

这让您能够更好地控制自动化工作流中的命令执行。

## 生活质量改进

* **代码库索引开关**：在设置中添加了代码库索引的启用/禁用复选框，状态在会话之间保持持久化（感谢 daniel-lxs、elasticdotventures！）
* **聊天界面**：改进了聊天布局，组织更清晰，任务历史可见性更高（显示 3 个任务而非 2 个）
* **命令消息提示**：当命令前缀被拒绝时提供更清晰的消息提示 - 用户现在可以看到拒绝是由于他们的设置，而非系统自动行为
* **历史记录导航**：在 HistoryPreview 组件中添加了“查看所有历史记录”链接，当历史记录标签页隐藏时，可更轻松地访问完整的任务历史记录

## 其他改进

* **类型定义**：更新了内部类型定义，以提高兼容性和开发体验
