# Roo Code 3.19.1 版本说明 (2025-05-31)

此补丁版本提供了关键修复和实验性功能，旨在增强您的开发体验，包括多文件读取功能、改进的 MCP 服务器集成以及企业级安全增强。

---

## 多文件读取（实验性功能）
我们引入了一项实验性功能，以更高效地处理大型代码库：

- **批量文件处理**：单次操作最多可读取 100 个文件，实现全面的代码分析
- **性能控制**：可调节的并发文件读取限制，以优化您的系统性能
- **智能权限管理**：简化多文件操作的审批流程
- **效率提升**：显著减少分析多个相关文件所需的时间

此实验性功能彻底改变了 Roo Code 一次性理解和处理整个代码库的方式（感谢 samhvw8！）。

---

## 企业级 VPC 终端节点支持
我们为 [AWS Bedrock](/providers/bedrock) 用户添加了关键的企业级安全功能：

- **安全的企业访问**：配置自定义 VPC 终端节点，确保所有 LLM 事务都在您的防火墙内进行
- **简单配置**：通过简单的复选框和文本字段即可启用 VPC 终端节点使用
- **合规就绪**：在使用强大 AI 模型的同时满足您组织的安全策略
- **灵活控制**：根据需要开启/关闭 VPC 终端节点，并持久化保存设置

企业客户现在可以在保持完全网络安全的同时，使用 Roo Code 与 AWS Bedrock（感谢 kcwhite！）。有关设置说明，请参阅 [AWS Bedrock 配置指南](/providers/bedrock#configuration-in-roo-code)。
