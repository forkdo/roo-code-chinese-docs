# Roo Code 3.10.2 版本说明 (2025-03-21)

此补丁版本包含对上下文提及、国际化和模型 token 限制的修复。

---

## Bug 修复

*   解决了 Windows 上上下文提及的问题。
*   更正了德语翻译。（感谢 cannuri！）
*   修复了遥测横幅的国际化问题。
*   确保 Sonnet 3.7 非思考模式正确使用 8192 个最大输出 token。
