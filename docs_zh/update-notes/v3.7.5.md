# Roo Code 3.7.5 发布说明 (2025-02-26)

本次发布包含对思维模型配置的重要更新和错误修复。

---

## 模型配置更新

*   **思维模型版本：** 为 Anthropic 和 OpenRouter Sonnet 3.7 模型引入了独立的 `:thinking` 版本，以支持可配置的思维预算。此前使用思维功能的用户需要在提供商设置中选择这些新模型版本，并根据需要调整思维预算滑块。

---

## 错误修复

*   修复了上下文窗口计算错误（“输入长度和 max_tokens 超出上下文限制”）。
*   修复了模型选择器 UI 的多个问题。 (感谢 System233!)
*   修复了模型输入/输出成本解析问题。 (感谢 System233!)

---

## 功能亮点

*   新增功能：按住 Shift 键的同时将文件从文件资源管理器拖入聊天输入框，即可 @-提及文件。
