# Roo Code 3.13 版本说明 (2025-04-18)

此发布周期带来了 UI 增强、新的文件操作工具、扩展的 Gemini 模型支持（包括 Flash 和自定义 URL），以及各种错误修复。

---

## 提供商更新

*   为 Gemini 和 Vertex 提供商添加了对 Gemini 2.5 Flash 预览版的支持（感谢 @nbihan-mediware！）。
*   启用了 Gemini 2.5 Flash 思考模式（感谢 @monotykamary）。
*   允许为 Gemini 提供商使用自定义 URL。

---

## 生活质量改进

*   对任务标题、聊天视图、历史记录预览和欢迎视图进行了 UI 改进（感谢 @sachasayan！）。
*   使自动批准切换的开关状态更加明显（感谢 @sachasayan）。

---

## 新工具

*   添加了 `append_to_file` 工具，用于向文件追加内容（感谢 @samhvw8！）。

---

## 错误修复

*   修复了 Bedrock 中的图像支持（感谢 @Smartsheet-JB-Brown！）。
*   使差异编辑对模型传入错误参数的情况更具弹性。
*   修复了 Windows 上将文件拖入聊天文本区域时的路径问题（感谢 @NyxJae）。

---

## 其他改进

*   添加了用于 shell 集成错误的遥测功能。
