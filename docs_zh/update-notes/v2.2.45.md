# Roo Code 2.2.45 版本说明

此版本引入了 API 配置配置文件功能。

---

## 功能亮点

*   **API 配置配置文件：** 新增将不同 API 配置（提供程序、模型、API 密钥、其他设置）保存为命名配置文件的功能，可在不同设置之间快速切换。（感谢 samhvw8！）
