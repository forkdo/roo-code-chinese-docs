# Roo Code 3.8.1 版本说明 (2025-03-07)

此补丁版本包含 UI 改进、错误修复以及遥测功能的增强。

---

## 改进与修复

*   在上下文窗口可视化中显示预留的输出 token。
*   改进了配置配置文件下拉菜单的 UI。（感谢 DeXtroTip！）
*   修复了无法取消勾选自定义温度的错误。（感谢 System233！）
*   修复了无法为 OpenAI 兼容提供商输入小数价格的错误。（感谢 System233！）
*   修复了 Sonnet 3.7 在高思考预算下增强提示功能的错误。（感谢 moqimoqidea！）
*   修复了思考模型上下文窗口管理的错误。（感谢 ReadyPlayerEmma！）
*   修复了检查点默认不再启用的错误。
*   在遥测数据中添加了扩展和 VSCode 的版本信息。
