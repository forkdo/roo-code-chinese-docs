# Roo Code 3.3.12 发布说明

此补丁版本新增了 Gemini 模型，并修复了模式配置相关的问题。

---

## 更新与修复

*   新增 Gemini 2.0 模型（Flash、Flash Lite Preview、Pro Experimental）。
*   修复了在“Prompts”选项卡中更改模式的 API 配置时出现的错误。
