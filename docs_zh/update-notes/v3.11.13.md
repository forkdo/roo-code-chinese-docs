# Roo Code 3.11.13 版本发布说明（2025-04-11）

此版本包含多项终端增强功能、改进的差异错误显示、文件上下文跟踪以及各类修复。

---

## 新增终端设置（感谢 KJ7LNW！）

新增六项可配置设置以提升终端可靠性：

*   **终端命令延迟设置** - 在运行命令后添加短暂暂停，以修复某些终端中的输出捕获问题。如果 Roo 难以查看命令结果，此功能会很有用。
*   **PowerShell 计数器规避设置** - 帮助 PowerShell 多次运行相同命令而不会失败。如果 Roo 在处理重复命令时遇到问题，请启用此功能。
*   **清除 ZSH 行尾标记设置** - 防止 ZSH 添加特殊字符，这些字符可能会干扰 Roo 读取终端输出。
*   **Oh My Zsh 集成设置** - 让 Roo 更好地与流行的 Oh My Zsh Shell 定制框架协同工作。（实验性功能）
*   **Powerlevel10k 集成设置** - 提升与 Powerlevel10k ZSH 主题的兼容性。（实验性功能）
*   **ZDOTDIR 处理设置** - 帮助 Roo 在不干扰个人设置的情况下使用自定义 ZSH 配置。（实验性功能）

有关这些设置和 Shell 集成故障排除的详细信息，请参阅[终端 Shell 集成](/features/shell-integration)。
