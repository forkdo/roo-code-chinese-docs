# Roo Code 3.0.0 发布说明

此重大版本引入了聊天模式！

---

## 功能亮点

*   **Chat Modes（聊天模式）：** 新增了不同的模式（Code、Architect、Ask），使您能够针对编码、系统设计或一般问题等特定任务定制交互。每种模式均可分配不同的 API 配置配置文件。
