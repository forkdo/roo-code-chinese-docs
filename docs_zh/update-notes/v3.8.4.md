# Roo Code 3.8.4 版本说明 (2025-03-09)

此补丁版本包含一项暂时回滚和一项新的提示选项。

---

## 更新内容

*   暂时回滚了多差异进度指示器，以修复保存编辑时的双重确认问题。
*   在 Prompts 选项卡中添加了禁用创建/编辑自定义模式说明的选项，以节省令牌。（感谢 hannesrudolph！）
