# Roo Code 3.10.0 发布说明 (2025-03-20)

此版本引入了建议回复、改进的大文件处理和增强的 @-提及功能。

---

## 功能亮点

*   **建议回复：** 在 Roo 提问时添加了快速回复选项。从列表中选择，而无需全部手动输入。(感谢 samhvw8!)
*   **大文件支持：** 现在通过分块加载使大文件读取更加高效，允许处理以前会导致上下文问题的文件。(感谢 samhvw8!)
*   **改进的 @-提及：** 完全重新设计了 @-提及的文件和文件夹查找功能，使用服务器端处理并支持 gitignore，在引用工作区文件时获得更准确的结果。

---

## Bug 修复和改进

*   使建议回复变为可选，以防止与覆盖的系统提示冲突。
*   修复了 MCP 错误日志记录。(感谢 aheizi!)
*   修复了 GitHub Releases 中的变更日志格式。(感谢 pdecat!)
*   修复了在使用 WSL 时导致任务历史丢失的 bug。
*   将代码操作整合到子菜单中。(感谢 samhvw8!)
*   改进了 `search_files` 工具的格式和逻辑。(感谢 KJ7LNW!)
*   为集成测试添加了假提供程序。(感谢 franekp!)
*   在 ap-xx 区域中反映了跨区域推理选项。(感谢 Yoshino-Yukitaro!)
