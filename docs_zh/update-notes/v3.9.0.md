# Roo Code 3.9.0 发布说明（2025-03-18）

本次发布新增了国际化支持、通过 SSE 实现的远程 MCP 连接、文本转语音功能，以及多项其他改进。

---

## 功能亮点

*   **国际化：** Roo Code 现在支持 14 种语言：简体中文、繁体中文、西班牙语、印地语、法语、葡萄牙语、德语、日语、韩语、意大利语、土耳其语、越南语、波兰语和加泰罗尼亚语。可在“高级设置” > “语言”中更改语言。（感谢 feifei325！）
*   **MCP 远程连接（SSE）：** 新增通过服务器发送事件（SSE）连接远程 MCP 服务器的支持，补充了现有的 stdio 支持。（感谢 aheizi！）
*   **文本转语音：** 新增选项，允许 Roo 在提供视觉响应的同时提供音频反馈。可在“高级设置” > “通知”中启用。（感谢 heyseth！）
*   **OpenRouter 提供商选择：** 新增在使用 OpenRouter 模型时，通过“配置配置文件”设置选择特定提供商的功能。（感谢 PhunkyBob！）
*   **批量历史记录删除：** 新增通过“查看全部”历史记录界面选择并一次性删除多个任务历史记录项的支持。（感谢 aheizi！）

---

## 终端改进

*   通过“高级设置”将终端 shell 集成超时时间设为可配置（1-60 秒），以解决 shell 启动时间过长的问题。（感谢 filthy、kiwina、KJ7LNW！）
*   修复了可能导致终端输出挂起或无法识别的竞态条件。（感谢 KJ7LNW！）
