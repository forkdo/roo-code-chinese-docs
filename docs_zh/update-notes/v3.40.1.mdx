# Roo Code 3.40.1 版本说明 (2026-01-14)

此版本修复了使用 Gemini 时的模式切换错误。

* 修复了在使用 Gemini 时切换模式（例如从 Code 切换到 Architect）会导致 API 错误的问题，该问题是由于对话历史中的工具权限冲突引起的。 ([#10708](https://github.com/RooCodeInc/Roo-Code/pull/10708))
