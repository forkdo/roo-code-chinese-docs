# Roo Code 3.3.21 版本说明 (2025-02-17)

此版本引入了 `@terminal` 提及功能，并包含多项改进和修复。

---

## 功能亮点

*   **@terminal 提及：** 在聊天输入中添加提及 `@terminal` 的功能，可直接将最近的终端输出拉入上下文。（感谢 Cline！）

---

## 常规改进与修复

*   为 OpenAI `o1` 模型启用了流式传输模式。
*   修复了系统提示，确保 Roo 了解所有可用模式。
*   修复了 `zh-cn` 和 `zh-tw` 的默认首选语言设置。（感谢 System233！）
*   修复了输入框还原问题以及配置文件切换期间的配置丢失问题。（感谢 System233！）
*   修复了 Mistral 提供商集成问题。（感谢 d-oit！）
