# Roo Code 3.23.15 Release Notes (2025-07-18)

此版本改进了诊断处理、UI 可访问性，并包含各种生活质量改进和错误修复。

## QOL 改进

* **任务安全**: 新设置可防止意外完成包含未完成待办事项的任务 ([#5716](https://github.com/RooCodeInc/Roo-Code/pull/5716))
* **Go 诊断**: 可配置延迟可防止关于未使用导入的错误报告 (感谢 mmhobi7!) ([#5863](https://github.com/RooCodeInc/Roo-Code/pull/5863))
* **市场访问**: 市场图标移至顶部导航以方便访问 ([#5864](https://github.com/RooCodeInc/Roo-Code/pull/5864))
* **自定义模式**: 为自定义模式添加了有用的描述和使用指导 (感谢 RandalSchwartz!) ([#5862](https://github.com/RooCodeInc/Roo-Code/pull/5862))
* **YouTube 页脚**: 从网站快速访问 Roo Code 的 YouTube 频道 (感谢 thill2323!) ([#5929](https://github.com/RooCodeInc/Roo-Code/pull/5929))
* **PR 模板**: 问题修复模式现在使用官方的 Roo Code PR 模板 (感谢 MuriloFP!) ([#5839](https://github.com/RooCodeInc/Roo-Code/pull/5839))

## 错误修复

* **Claude 模型检测**: 修复了通过 Google Vertex AI 等替代提供商使用 Claude 模型时的上下文长度错误 (感谢 daniel-lxs, TheFynx!) ([#5840](https://github.com/RooCodeInc/Roo-Code/pull/5840))

## 其他改进

* **命令超时配置**: 为长时间运行的命令添加了命令超时白名单 ([#5910](https://github.com/RooCodeInc/Roo-Code/pull/5910))
* **内部模式**: 为内部 GitHub 工作流添加了问题调查模式并增强了问题编写模式 ([#5913](https://github.com/RooCodeInc/Roo-Code/pull/5913))
