# Roo Code 3.3.18 版本发布说明 (2025-02-11)

此版本引入了模型温度控制、Requesty 提供商支持，以及各种用户体验改进和错误修复。

---

## 功能亮点

*   **温度控制：** 为每个 API 配置添加了模型温度设置，允许根据模式为同一模型设置不同的创造性级别。（感谢 joemanley201！）
*   **Requesty 提供商支持：** 添加了对 Requesty 提供商的支持。（感谢 samhvw8！）

---

## 用户体验改进

*   在“提示”标签页中为系统提示添加了复制按钮。（感谢 mamertofabian！）

---

## 错误修复

*   添加了获取 OpenRouter 使用统计信息的重试机制。（感谢 jcbdev！）
*   修复了 MCP 服务器在启动时有时不会在设置中显示的问题。（感谢 MuriloFP！）
*   修复了 Ollama/LMStudio 设置在设置中闪烁的问题。
*   修复了不正确的 API 重试时间计算。
*   修复了 Windows 上的检查点问题。（感谢 CTE！）
