此补丁版本增加了对 OpenAI 模型的支持，并改进了 UI 和遥测功能。

*   新增对 OpenAI `o3` 和 `4o-mini` 模型的支持。（感谢 PeterDaveHello！）

## 改进

*   改进了文件/文件夹上下文提及 UI，提升了可用性。（感谢 elianiva！）
*   增强了差异错误遥测功能，提高了故障排查能力。
