# Roo Code 3.25.23 版本说明 (2025-08-22)

此版本为 Requesty 添加了自定义基础 URL 支持，并为 Chutes AI 添加了 DeepSeek V3.1 模型。

## 提供商更新

* **Requesty Provider**：添加了对自定义基础 URL 的支持，允许您连接到自托管或替代的 Requesty 端点 ([#7337](https://github.com/RooCodeInc/Roo-Code/pull/7337))
* **Chutes AI**：添加了对 DeepSeek V3.1 模型的支持，扩展了 Chutes AI 用户可用的模型选项 ([#7295](https://github.com/RooCodeInc/Roo-Code/pull/7295))
