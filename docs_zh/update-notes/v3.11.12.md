# Roo Code 3.11.12 发布说明 (2025-04-09)

此版本支持通过 OpenAI Compatible 提供商启用 Grok3 流式传输，并改进了 diff 编辑的容错性。

---

## 提供商更新

*   使 Grok3 流式传输能够与 OpenAI Compatible 配合使用（感谢 amittell！）

---

## 改进

*   调整 diff 编辑逻辑，使其对模型错误更加宽容
