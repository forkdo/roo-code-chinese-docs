# Roo Code 3.3.7 发布说明

此版本新增了对 `o3-mini` 模型的支持，改进了 Code Actions 和工具交互功能，并引入了 API 限流机制。

---

## 功能亮点

*   新增对 `o3-mini` 模型的支持。（感谢 shpigunov！）
*   改进 Code Actions：新增选择代码并直接添加到上下文的功能，并修复了若干问题。（感谢 samhvw8！）
*   在批准或拒绝工具使用时，可添加附带消息。（感谢 napter！）
*   新增 API 重试的指数退避策略。
*   在高级设置中新增可选滑块，用于启用 API 请求限流（例如：请求之间至少等待 X 秒）。

---

## 改进与修复

*   改进了聊天输入框的样式。（感谢 psv2522！）
*   增强了对 DeepSeek R1 更多变体的推理捕获能力。（感谢 Szpadel！）
*   调整提示词，以提升 Roo 创建新自定义模式的能力。
