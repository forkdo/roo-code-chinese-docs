# Roo Code 3.11.2 发布说明 (2025-03-31)

此补丁版本包含错误修复和内部改进。

---

## 错误修复

*   修正了阻止准确加载 Requesty API 密钥余额的问题。
*   解决了影响 Bedrock 推理配置文件使用的问题。
*   确保通过 API 修改设置时 webview 正确更新。
---

## 通用和生活质量改进

*   重构了内部 webview 消息处理，以改进结构和可维护性（感谢 @diarmidmackenzie！）。
