# Roo Code 3.8.5 发布说明（2025-03-12）

本次发布带来了架构重构、远程浏览器支持、通过 SSE 实现的 MCP 通信，以及多个提供商和模型的更新。

---

## 功能亮点

*   **终端架构重构：** 解决了终端架构中的关键问题，提升了稳定性和性能。 (感谢 KJ7LNW！)
*   **MCP over SSE：** 新增对通过服务器发送事件（SSE）实现模型上下文协议（MCP）通信的支持。 (感谢 aheizi！)
*   **远程浏览器支持：** 新增连接远程 Chrome 浏览器实例的功能。 (感谢 afshawnlotfi！)

---

## 提供商与模型支持

*   为 Google AI Studio Gemini 添加了自定义 `baseUrl` 支持。 (感谢 dqroid！)
*   在兼容 OpenAI 的提供商中新增了 DeepSeek/QwQ 推理支持。 (感谢 lightrabbit！)
*   在兼容 OpenAI 的提供商中新增了类似 Anthropic 的提示缓存功能。 (感谢 dleen！)
*   为 AWS Bedrock 添加了 Deepseek R1 模型。 (感谢 ATempsch！)
*   在 Vertex AI 提供商中新增了 `gemini-2.0-pro-exp-02-05` 模型。 (感谢 shohei-ihaya！)
*   为 AWS Bedrock 添加了对自定义 ARN 的支持。 (感谢 Smartsheet-JB-Brown！)
*   更新了 Bedrock 的定价信息至最新版本。 (感谢 Smartsheet-JB-Brown！)

---

## 改进与修复

*   在取消 Boomerang 任务（子任务）时保留了父子关系。 (感谢 cannuri！)
*   新增了 PowerShell 特定命令处理功能。 (感谢 KJ7LNW！)
*   修复了 Dark High Contrast 主题下 MarkdownBlock 文本颜色问题。 (感谢 cannuri！)
*   恢复了多差异编辑的进度状态指示器。 (感谢 qdaxb！)
*   重构了警告对话框样式以使用正确的 VSCode 主题。 (感谢 cannuri！)
*   更新了 MCP 服务器目录路径以提升平台兼容性。 (感谢 hannesrudolph！)
*   修复了浏览器系统提示的包含规则。 (感谢 cannuri！)
*   修复了 OpenAI 风格的成本计算问题。 (感谢 dtrugman！)
*   修复了允许将已排除目录用作工作目录的问题。 (感谢 Szpadel！)
*   在 `list_code_definition_names` 工具中新增了 Kotlin 语言支持。 (感谢 kohii！)
*   改进了差异应用错误的处理机制。 (感谢 qdaxb！)
*   修复了 OpenRouter 的自定义 `baseUrl` 支持。
*   修复了 SiliconFlow 及其他在每个数据块中包含使用情况的提供商的使用量追踪。
*   新增了对检查点保存/恢复/差异以及差异策略的遥测功能。
*   从 CI 发布 git 标签至 GitHub。 (感谢 pdecat！)
