# Roo Code 3.7.7 版本说明 (2025-02-27)

此版本正式发布 Checkpoints 功能，并包含一些小的修复和用户体验调整。

---

## 功能亮点

*   **默认启用 Checkpoints：** Checkpoints 功能现在默认启用，可在任务期间自动跟踪项目变更，以便审查或回滚。可在高级设置中禁用此功能。

---

## Bug 修复和用户体验调整

*   修复了使用 Thinking Sonnet 模型时“增强提示”按钮的功能问题。
*   为多个按钮添加了工具提示，以提高清晰度。
