# Roo Code 3.10.3 发布说明（2025-03-23）

本次发布重点改进了大文件处理，并包含多项错误修复。

---

## 功能亮点

*   增强了部分文件读取功能，支持显式请求完整文件读取、自定义分块大小，并提供更清晰的操作指引。

---

## 一般性改进

*   更新了欢迎页面，提供与 LLM 路由器的一键 OAuth 流程。（感谢 dtrugman！）
*   改为使用更直接的方式跟踪 OpenRouter 代币/支出。

---

## 错误修复

*   修复了非流式模型中未显示问题与建议，以及在某些主题下难以阅读的问题。
*   解决并改进了实验性多块差异功能的相关问题。（感谢 KJ7LNW！）
*   修复了设置中下拉菜单的透明度问题。（感谢 KJ7LNW！）
*   修复了读取和提及 PDF 等二进制文件时的相关问题。
*   更正了 OpenRouter 免费模型的定价信息。（感谢 Jdo300！）
*   修复了 Windows 上单元测试的问题。（感谢 diarmidmackenzie！）
*   解决了 Outbound 提供商的 `maxTokens` 问题。（感谢 pugazhendhi-m！）
*   修复了与部分文件读取相关的行号问题。（感谢 samhvw8！）
