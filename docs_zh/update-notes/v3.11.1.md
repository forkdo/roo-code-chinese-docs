# Roo Code 3.11.1 发布说明（2025-03-30）

此补丁版本包含对提供程序配置的少量更新。

---

## 通用改进与质量提升

*   **提供程序配置文件模式：** 已放宽对提供程序配置文件的 JSON 模式验证。这为配置自定义或较少见的提供程序设置提供了更大的灵活性，并减少了对有效配置的潜在验证错误。同时已添加与提供程序使用情况相关的遥测数据，以帮助诊断问题。
