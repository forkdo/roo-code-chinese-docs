# Roo Code 3.3.20 发布说明 (2025-02-14)

本次发布新增了项目级自定义模式，并对 Ask 模式进行了更新。

---

## 功能亮点

*   **项目级自定义模式：** 支持在工作区内的 `.roomodes` 文件中定义项目特定的自定义模式。
*   **Ask 模式更新：** Ask 模式现已专注于聊天交互，不再支持编辑 Markdown 文件。

---

## 提供程序支持

*   新增了 Mistral 模型支持。（感谢 d-oit、bramburn！）

---

## 一般改进

*   新增设置项，用于控制上下文中包含的可见编辑器标签数量。
*   通过修复欢迎界面中的 API 密钥输入问题，改进了初始设置体验。
