# Roo Code 3.12.3 版本发布说明 (2025-04-17)

此版本包含对 Gemini 差异编辑和斜杠命令的修复，以及一项新的拖放功能。

---

## 错误修复

*   修复了 Gemini 差异编辑中的字符转义问题。
*   确保斜杠命令仅在聊天框开头触发（感谢 logosstone！）。

---

## 体验优化

*   添加了对将标签页拖放到聊天框的支持（感谢 NyxJae！）。
