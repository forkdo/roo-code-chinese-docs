# Roo Code 3.3.0 版本说明

此版本引入了原生 Code Actions、更智能的模式切换、针对特定模式的增强 Markdown 支持，以及针对 Bedrock 的 AWS 配置文件支持。

---

## 功能亮点

*   **Code Actions 支持：** 集成了原生 VS Code 代码操作，可直接在编辑器中提供快速修复和重构建议。
*   **更智能的模式切换：** 模式现在可以在适当时智能地请求切换到其他模式（例如，当从 Architect 模式需要编辑代码时，建议切换到 Code 模式）。
*   **增强的 Markdown 支持：** Ask 和 Architect 模式现在支持编辑 Markdown 文件。
*   **自定义文件模式限制：** 新增功能，可将自定义模式限制为特定的文件模式（例如，允许“技术文档撰写者”模式仅编辑 `.md` 文件）。
*   **Bedrock 的 AWS 配置文件：** 新增支持使用 AWS 配置文件配置 Bedrock 提供商，有利于 SSO 或无长期凭证的集成。
