# Roo Code 3.28.18 版本发布说明 (2025-10-17)

此补丁修复了灰色屏幕问题，优化了图像生成的默认设置，并更新了提供商配置。


## 灰色屏幕死机问题修复

这应该能修复由长时间上下文任务会话引起的灰色屏幕问题 ([#8696](https://github.com/RooCodeInc/Roo-Code/pull/8696))。

## 生活质量改进

* 图像生成模型选择更新：默认使用 Gemini 2.5 Flash Image；新增 OpenAI GPT‑5 Image 和 GPT‑5 Image Mini；设置下拉菜单更加清晰（感谢 chrarnoldus！） ([#8698](https://github.com/RooCodeInc/Roo-Code/pull/8698))

## Bug 修复

* 编辑器定位修复：避免编辑只读的 git diff 视图；改为编辑实际文件（感谢 hassoncs！） ([#8676](https://github.com/RooCodeInc/Roo-Code/pull/8676))

## 提供商更新

* Bedrock：添加带版本信息的用户代理，用于按版本进行指标统计和错误追踪；无需任何操作（感谢 ajjuaire！） ([#8663](https://github.com/RooCodeInc/Roo-Code/pull/8663))
* Z AI：仅支持两个编码端点（国际版/中国版）；默认为国际版；不支持旧版非编码端点 ([#8693](https://github.com/RooCodeInc/Roo-Code/pull/8693))
