# Roo Code 3.23.5 版本说明 (2025-07-09)

此版本修复了文件处理问题并提升了账户可访问性。

## 错误修复

* **错误消息**: 修复了嵌入提供程序错误消息中原始翻译键的显示问题 - 现在会正确显示本地化消息 (感谢 daniel-lxs!) ([#5535](https://github.com/RooCodeInc/Roo-Code/pull/5535))
* **文件打开**: 修复了打开文件名和文件夹名中包含括号等特殊字符的文件的功能 (感谢 vivekfyi, nielpattin!) ([#5504](https://github.com/RooCodeInc/Roo-Code/pull/5504))

## 体验优化

* **账户访问**: 账户功能现在更易访问 - 账户标签页显示在主导航中，而不是隐藏在溢出菜单中 ([#5534](https://github.com/RooCodeInc/Roo-Code/pull/5534))
