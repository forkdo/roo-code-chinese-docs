# Roo Code 3.13.0 版本说明 (2025-04-17)

此版本带来了多个视图的显著 UI 改进，新增了 append_to_file 工具，引入了 Gemini 2.5 Flash Preview 支持，并包含了重要的错误修复。

---

## Gemini 2.5 Flash Preview
- 在 Gemini 和 Vertex 提供商中添加 Gemini 2.5 Flash Preview（感谢 nbihan-mediware！）

---

## UI 改进
- 对任务标题、聊天视图、历史预览和欢迎视图进行 UI 改进（感谢 sachasayan！）

---

## 新工具：append_to_file
- 新增 append_to_file 工具，用于向文件追加内容（感谢 samhvw8！）
  - 高效地向现有文件末尾添加内容或创建新文件
  - 适用于日志、数据记录和增量文件构建
  - 包含自动目录创建和通过差异视图进行交互式审批
  - 通过专门的追加功能补充现有的文件操作工具

---

## 错误修复
- 修复 Bedrock 中的图像支持（感谢 Smartsheet-JB-Brown！）
- 使差异编辑对模型传入错误参数的情况更具弹性
