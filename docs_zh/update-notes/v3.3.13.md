# Roo Code 3.3.13 版本发布说明

此版本包含提供商支持更新、终端改进以及错误修复。

---

## 改进与修复

*   确保 DeepSeek R1 模型与 Ollama 正确兼容。（感谢 sammcj！）
*   在 Roo Code 使用的集成终端中启用了上下文菜单命令（如复制/粘贴）。（感谢 samhvw8！）
*   改进了不支持提示缓存的模型的滑动窗口截断策略。（感谢 nissa-seru！）
*   针对切换 API 配置文件相关的错误实施了初步修复（计划进一步改进）。
