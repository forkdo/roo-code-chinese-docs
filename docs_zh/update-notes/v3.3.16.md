# Roo Code 3.3.16 版本说明 (2025-02-09)

此版本包含对检查点和 API 配置的修复，以及对 Volcano Ark 的支持。

---

## Bug 修复与改进

*   通过失焦更新而非输入更新，修复了进入 API 配置时的跳转问题。
*   为检查点操作添加了工具提示。
*   修复了检查点覆盖现有 git 用户名/邮箱设置的问题。

---

## 服务提供商支持

*   通过 OpenAI 兼容提供商添加了对 Volcano Ark 平台的支持。
