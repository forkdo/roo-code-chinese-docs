# Roo Code 3.11.8 发行说明 (2025-04-05)

此补丁版本包含性能改进和 UI 更新。

---

## 引入 `.roorules` 文件

*   新增对 `.roorules` 文件的支持（例如，工作区范围的 `.roorules`，模式特定的 `.roorules-code`），作为在项目中直接管理自定义指令的一种方式。这为仅通过扩展设置 UI 定义指令提供了替代方案。同时添加了对旧版 `.clinerules` 文件的弃用警告。（感谢 upamune！）
*   了解更多关于 [Custom Instructions](/features/custom-instructions) 以及其在 [Custom Modes](/features/custom-modes) 中的应用。
---

## 变更

*   改进了 `combineApiRequests` 的性能，以减少“灰屏死机”现象（感谢 kyle-apex！）。
*   在设置屏幕的 API 配置配置文件中添加了可搜索的下拉菜单（感谢 samhvw8！）。
*   为历史记录项添加了工作区跟踪，为未来的筛选功能做准备（感谢 samhvw8！）。
---

## 问题修复

*   修复了历史记录搜索中的高亮显示 UI（感谢 samhvw8！）。
*   修复了 `.tool-versions` 文件中的 nodejs 版本格式（感谢 upamune！）。
