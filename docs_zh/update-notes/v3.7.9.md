# Roo Code 3.7.9 版本说明（2025-03-01）

此版本包含更智能的上下文管理、终端解析改进以及 UI 优化。

---

## 改进与修复

*   实现了更智能的上下文窗口管理，以减少上下文限制错误。
*   改进了终端输出解析逻辑，以解决影响命令输出可见性的 VSCode 问题。（感谢 KJ7LNW！）
*   增加了通过 Vertex AI 对 Claude Sonnet 3.7 思考模式的支持。（感谢 lupuletic！）
*   修复了 Claude 3.7 Sonnet 模型的 `maxTokens` 默认值。
*   修复了 UI 下拉菜单悬停颜色。（感谢 SamirSaji！）
*   优化了思考块的显示效果。
*   增强了删除任务确认对话框。
