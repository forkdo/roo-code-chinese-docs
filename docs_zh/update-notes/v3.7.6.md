# Roo Code 3.7.6 发布说明（2025-02-26）

本次发布新增了多文件拖放功能、为思考模型添加了可配置的输出 tokens，并修复了若干问题。

---

## 功能亮点

*   新增支持将多个文件拖放到聊天输入框中。
*   新增滑块，用于控制思考模型的输出 tokens 最大值。

---

## 问题修复

*   改进了对聊天消息中极长文本的处理。（感谢 joemanley201！）
*   截断 `search_files` 输出，防止可能导致的扩展崩溃。
*   改进了 OpenRouter 错误处理（已替换通用的“Provider Error”）。
