# Roo Code 2.2.29 发布说明

本次发布新增了对自动写入后延迟的配置支持。

---

## 通用改进与质量提升

*   添加了在自动批准的文件写入后可配置的延迟，以便诊断工具（如 linter）有时间处理更改。
