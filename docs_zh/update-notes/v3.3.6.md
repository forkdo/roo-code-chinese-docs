# Roo Code 3.3.6 版本说明

此版本引入了 `new_task` 工具、UI 改进以及新的提供商支持。

---

## 功能亮点

*   **新任务工具：** 添加了 `new_task` 工具，允许 Roo 通过初始消息和模式以编程方式启动新任务，从而实现上下文延续或记忆库更新等工作流。

---

## UI 改进

*   增强了下拉菜单的视觉效果，以提供更好的用户体验。（感谢 psv2522！）

---

## 提供商支持

*   新增对 Perplexity Sonar Reasoning 模型的支持。（感谢 Szpadel！）
*   新增对 Unbound 提供商的支持。（感谢 vigneshsubbiah16！）

---

## Bug 修复

*   修复了影响 `qwen-max` 以及可能影响其他 OpenAI 兼容提供商的关键错误。（感谢 Szpadel！）
