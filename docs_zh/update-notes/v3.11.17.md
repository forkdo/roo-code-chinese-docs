# Roo Code 3.11.17 版本说明 (2025-04-14)

此版本包含对 OpenAI 缓存报告、UI 视觉效果、差异逻辑、终端命令捕获的改进，并修复了一个 eslint 错误。

---

## 改进

*   改进了 OpenAI 缓存报告和成本估算（感谢 monotykamary 和 Cline！）
*   自动批准切换的视觉改进（感谢 sachasayan！）
*   添加了遥测功能以跟踪未来的差异应用错误

---

## Bug 修复

*   修复了差异应用逻辑（感谢 avtc 提供的测试用例！）
*   修复了捕获短时运行终端命令时的竞态条件（感谢 KJ7LNW！）
*   修复了 eslint 错误（感谢 nobu007！）
