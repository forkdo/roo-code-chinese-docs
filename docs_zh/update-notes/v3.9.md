# Roo Code 3.9 版本发布说明 (2025-03-19)

此发布周期引入了广泛的国际化、文本转语音功能、通过 SSE 的 MCP 以及众多对翻译、UI 和错误修复的改进。

---

## 国际化与可访问性

*   **广泛的语言支持：** Roo Code 已支持加泰罗尼亚语、德语、西班牙语、法语、印地语、意大利语、日语、韩语、波兰语、葡萄牙语、土耳其语、越南语、简体中文和繁体中文。（感谢 @feifei325！）
*   **文本转语音：** 添加了让 Roo 在工作时与您对话的选项，包括保持语速状态。（感谢 @heyseth！）
*   **系统提示中的正确语言：** 确保当前语言正确传递到系统提示中。
*   **翻译修复：** 解决了法语翻译中的问题（感谢 @arthurauffray！），并修复了公告标题和浏览器的国际化。
*   **人工中继国际化：** 对人工中继功能进行了国际化，调整了其布局，并使其在欢迎屏幕上可用。（感谢 @NyxJae！）

---

## 提供商与 API 增强

*   **通过 SSE 的 MCP：** 重新引入了对通过服务器发送事件 (Server-Sent Events) 的 MCP（模式通信协议）的支持。（感谢 @aheizi！）
*   **OpenRouter 上的特定提供商：** 添加了在 OpenRouter 上选择特定提供商的功能。（感谢 PhunkyBob！）
*   **Bedrock 自定义 ARN 显示：** 修复了作为提示路由器的 Bedrock 自定义 ARN 的显示更新。（感谢 @Smartsheet-JB-Brown！）
*   **在 API 中公开任务堆栈：** 任务堆栈现在在 `RooCodeAPI` 中公开。（感谢 @franekp！）
*   **模型对 API 成本的可见性：** 模型现在可以查看当前任务的 API 成本。

---

## 生活质量改进

*   **批量任务历史删除：** 支持批量删除任务历史项。（感谢 @aheizi！）
*   **手动模型名称输入：** 恢复了在模型选择器中手动输入模型名称的功能。
*   **VSCode 主题修复：** 解决了选择、下拉和命令组件的主题不一致问题。

---

## 错误修复

*   **Shell 集成竞态条件：** 修复了 shell 集成中的竞态条件。（感谢 @KJ7LNW！）
*   **任务历史复制：** 修复了从任务历史复制项时排除搜索高亮显示的问题。（感谢 @im47cn！）
*   **多工作区上下文提及：** 上下文提及现在在多工作区项目中正常工作。（感谢 @teddyOOXX！）
*   **任务历史保存（多个 Roo）：** 修复了运行多个 Roo Code 实例时的任务历史保存问题。（感谢 @samhvw8！）
*   **删除缺失文件的任务：** 改进了当基础文件缺失时的任务删除。（感谢 @GitlyHallows！）
*   **NixOS 和 direnv 支持：** 改进了对 NixOS 和 direnv 环境的支持。（感谢 @wkordalski！）
*   **编辑器标签中的滚轮滚动：** 修复了当 Roo 在编辑器标签中打开时的滚轮滚动问题。（感谢 @GitlyHallows！）
*   **“添加到上下文”自动提及：** 防止使用“添加到上下文”代码操作时自动提及文件。（感谢 @qdaxb！）

---

## 其他改进

*   **GitHub 发布工作流：** 更新了 GitHub Actions 工作流以自动创建 GitHub 发布。（感谢 @pdecat！）
*   **优化构建时间：** 优化了本地开发的构建时间。（感谢 @KJ7LNW！）
