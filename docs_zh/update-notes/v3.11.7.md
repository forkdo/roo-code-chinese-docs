# Roo Code 3.11.7 版本发布说明 (2025-04-04)

此补丁版本包含对文件工具上下文、本地化以及内部处理的改进。

---

## 通用及体验优化改进

*   改进了文件工具上下文格式化和差异错误引导。
*   改进了 zh-TW 本地化（感谢 PeterDaveHello！）。
*   为 McpHub 资源释放实现了引用计数机制。
*   更新了按钮样式以提升一致性（感谢 kyle-apex！）。
*   改进了 zh-CN 本地化（感谢 System233！）。
