# Roo Code 3.7.10 版本说明 (2025-03-01)

此版本增加了对 Mermaid 图表的支持，扩展了模型选项，并引入了模式切换快捷键。

---

## 功能亮点

*   **Mermaid 图表支持：** 增加了在聊天对话中直接渲染 Mermaid 图表的功能。（感谢 Cline！）
*   **Vertex AI Gemini 模型：** 增加了 Vertex AI 提供商的 Gemini 模型。（感谢 ashktn！）
*   **模式切换快捷键：** 引入了用于在模式之间切换的键盘快捷键。点击模式弹出菜单可查看可用快捷键。（感谢 aheizi！）
