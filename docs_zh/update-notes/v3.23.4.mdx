# Roo Code 3.23.4 版本说明 (2025-07-09)

此版本包含错误修复、UI 改进和更好的组织。

## 错误修复

* **自定义模式**: 修复了当自定义模式配置文件为空或包含 null 内容时崩溃的问题 (感谢 daniel-lxs!) ([#5526](https://github.com/RooCodeInc/Roo-Code/pull/5526))
* **文件列表**: 修复了 [`list_files`](/advanced-usage/available-tools/list-files) 工具错误显示应被 .gitignore 模式忽略的目录的问题 (感谢 X9VoiD!) ([#5394](https://github.com/RooCodeInc/Roo-Code/pull/5394))

## 生活质量改进

* **聊天界面**: 改进了聊天界面按钮的工具提示一致性，提供更精致的使用体验 (感谢 daniel-lxs!) ([#5528](https://github.com/RooCodeInc/Roo-Code/pull/5528))
* **界面组织**: 更简洁的 Roo Code 视图标题，减少了视觉杂乱 - 不常用的操作移至 VS Code 的溢出菜单 (感谢 dlab-anton!) ([#3060](https://github.com/RooCodeInc/Roo-Code/pull/3060))
* **命令组织**: 更好地组织了 VS Code 扩展界面，命令在溢出菜单中正确分组 ([#5525](https://github.com/RooCodeInc/Roo-Code/pull/5525))
* **索引状态**: 改进了索引状态指示器的视觉对齐和一致性 ([#5523](https://github.com/RooCodeInc/Roo-Code/pull/5523))
* **界面图标**: 整个界面使用更一致、更清晰的图标，更好地显示代码库索引状态 ([#5520](https://github.com/RooCodeInc/Roo-Code/pull/5520))
