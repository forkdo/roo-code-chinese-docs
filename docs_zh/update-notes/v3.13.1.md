# Roo Code 3.13.1 版本发布说明 (2025-04-18)

此版本包含对 Gemini 2.5 Flash 思维模式的支持、自动审批切换的 UI 改进以及若干错误修复。

---

## Gemini 2.5 Flash 思维模式
- 支持 Gemini 2.5 Flash 思维模式（感谢 monotykamary！）

---

## UI 改进
- 使自动审批切换的开关状态更加明显（感谢 sachasayan！）

---

## 错误修复
- 修复在 Windows 系统中将文件拖入聊天文本区域时的路径问题（感谢 NyxJae！）
  - 解决了 Windows 环境下特有的路径规范化问题
  - 确保跨平台文件处理的一致性

---

## 遥测增强
- 为 Shell 集成错误添加遥测数据
