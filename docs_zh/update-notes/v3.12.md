# Roo Code 3.12 版本说明 (2025-04-17)

此发布周期引入了 xAI 提供商、Grok 推理选项、差异编辑增强功能、更快的检查点、UI 改进（如下拉菜单中的搜索栏）以及各种错误修复。

---

## 提供商更新

*   添加了 xAI 提供商，并公开了 OpenRouter 上 Grok 的推理工作量选项（感谢 Cline！）。
*   添加了 OpenAI o3 和 4o-mini（感谢 PeterDaveHello！）。

---

## 生活质量改进

*   将差异编辑配置设为每个配置文件独立，并改进了差异前字符串标准化。
*   使检查点更快、更可靠。
*   在模式和配置文件选择下拉菜单中添加了搜索栏（感谢 samhvw8！）。
*   在任务标题中隐藏零成本值（感谢 do-it！）。
*   允许用户绑定键盘快捷键以接受建议或输入聊天视图中的内容（感谢 axkirillov！）。
*   改进了文件/文件夹上下文提及 UI（感谢 elianiva！）。
*   添加了对将标签拖放到聊天框中的支持（感谢 NyxJae！）。

---

## 错误修复

*   修复了选择下拉菜单中编辑按钮可见性的错误。
*   修复了 Gemini 差异编辑中的字符转义问题。
*   确保斜杠命令仅在聊天框开头触发（感谢 logosstone！）。

---

## 其他改进

*   添加了代码操作使用情况、提示增强使用情况和连续错误错误的遥测功能。
*   使 JSON 解析更安全，避免在输入错误时崩溃 Webview。
*   改进了差异错误遥测功能。
