# Roo Code 3.8.0 版本发布说明 (2025-03-07)

此重要版本引入了 Boomerang Tasks、多块差异编辑、多窗口支持、`.rooignore`、Human Relay 提供商以及选择加入式遥测功能，同时包含大量改进和修复。

---

## 功能亮点

*   **Boomerang Tasks：** 使用 `new_task` 工具时，子任务现在会在完成后向父任务返回摘要，从而实现自动化交接和结果检索。（感谢 shaybc！）
*   **多块差异策略（实验性）：** 新增了一种实验性差异编辑策略，可同时应用多个差异编辑，可能实现更快、更复杂的更改。（感谢 qdaxb！）
*   **多窗口支持：** Roo Code 现在可以同时运行在多个 VS Code 编辑器窗口中。（感谢 samhvw8！）
*   **.rooignore 支持：** 新增对 `.rooignore` 文件的支持，可防止 Roo Code 读取/写入指定文件，还可选择将其从搜索/列表结果中排除。（感谢 mrubens、Cline！）
*   **Human Relay 提供商：** 引入新的提供商，当无法直接集成时，允许手动将信息复制到 Web AI 并将响应粘贴回 Roo Code。（感谢 NyxJae！）
*   **遥测（选择加入）：** 新增选择加入式遥测功能，用于收集匿名使用数据，帮助更快改进 Roo Code。可随时禁用。[隐私政策](https://github.com/RooCodeInc/Roo-Code/blob/main/PRIVACY.md)。（感谢 mrubens、Cline！）

## 用户体验改进

*   重新设计了设置页面，便于导航。（感谢 cte！）
*   将检查点设为异步并排除更多文件以加快其速度。（感谢 cte！）
*   改进了聊天中模式/提供商选择器的 UI。（感谢 mrubens！）
*   改进了任务标题的样式。（感谢 monotykamary！）

## 提供商支持

*   为 Vertex AI 添加了基于凭据的身份验证，便于在 Google Cloud 账户之间轻松切换。（感谢 eonghk！）
*   使用正确的 `baseUrl` 更新了 DeepSeek 提供商并修复了缓存跟踪。（感谢 olweraltuve！）
*   为 OpenAI 提供商添加了可观测性。（感谢 refactorthis！）
*   支持 LM Studio 本地模型的推测性解码。（感谢 adamwlarson！）
