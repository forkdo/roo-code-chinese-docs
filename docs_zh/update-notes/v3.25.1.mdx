# Roo Code 3.25.1 版本说明 (2025-07-29)

此版本为 Chutes AI 提供商添加了一个强大的新 AI 模型，并提高了命令执行的可靠性。

## 提供商更新

* **Chutes AI - GLM-4.5-Air 模型**：为 Chutes AI 提供商添加了对 GLM-4.5-Air 模型的支持，具有 151K 令牌的上下文窗口，可用于复杂推理任务和大代码库分析 - 完全免费使用（感谢 matbgn！）([#6377](https://github.com/RooCodeInc/Roo-Code/pull/6377))

## 生活质量改进

* **子 shell 验证**：改进了对带有子 shell 的复杂 shell 命令的处理和验证，防止在使用命令替换模式时出现潜在错误 ([#6379](https://github.com/RooCodeInc/Roo-Code/pull/6379))
