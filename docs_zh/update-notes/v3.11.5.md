# Roo Code 3.11.5 发布说明（2025-04-03）

此补丁版本新增了 Bedrock 提示缓存、可配置的 MCP 工作目录、API 配置文件管理以及差异编辑功能的改进。

---

## 通用与体验优化

*   为 Amazon Bedrock 添加了提示缓存功能（感谢 Smartsheet-JB-Brown！）。
*   新增对 MCP 服务器当前工作目录（CWD）配置的支持（感谢 shoopapa！）。
*   为 API 添加了配置文件管理功能（感谢 gtaylor！）。
*   改进了差异编辑功能、测试及错误提示（感谢 p12tic！）。

---

## 错误修复

*   修复了后续问题获取焦点的问题（感谢 diarmidmackenzie！）。
*   在将扩展程序弹出到新标签页时显示菜单按钮（感谢 benny123tw！）。
