# Roo Code 3.3.19 版本说明（2025-02-12）

此版本包含与文件写入、UI 主题、自定义指令和检查点相关的错误修复和改进。

---

## 错误修复与改进

*   修复了中止文件写入时不会还原更改的错误。
*   确保对话框背景遵循 VS Code 主题。
*   现在可以清除内置模式的默认自定义指令。
*   添加了链接到文档站点的帮助按钮。
*   将检查点逻辑切换为使用影子 Git 仓库，以避免热重载时出现问题以及污染现有仓库。（感谢 Cline！）
