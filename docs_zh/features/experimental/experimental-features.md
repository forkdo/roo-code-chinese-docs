# 实验性功能

Roo Code 包含一些仍处于开发阶段的实验性功能。这些功能可能不稳定、会发生显著变化，或在未来的版本中被移除。请谨慎使用，并注意它们可能无法按预期工作。

**警告：** 实验性功能可能会出现意外行为，包括潜在的数据丢失或安全漏洞。启用这些功能需自行承担风险。

---

## 启用实验性功能

要启用或禁用实验性功能：

1.  打开 Roo Code 设置（右上角的 `<Codicon name="gear" />` 图标）。
2.  进入“高级设置”部分。
3.  找到“实验性功能”部分。

---

## 当前实验性功能

当前可用的实验性功能包括：

- [并发文件编辑](/features/experimental/concurrent-file-edits) - 在单个操作中编辑多个文件
- [动力转向](/features/experimental/power-steering) - 增强 AI 响应的一致性
- [后台编辑](/features/experimental/background-editing) - 在 Roo 后台编辑文件时不受干扰地工作
- [图像生成](/features/image-generation) - 从文本提示生成图像并保存到工作区
- [运行斜杠命令](/advanced-usage/available-tools/run-slash-command) - 执行预定义的斜杠命令，用于模板化指令和工作流自动化
