# 实验性功能

Roo Code 包含仍在开发中的实验性功能。这些功能可能不稳定，可能会发生重大变化，或者在未来版本中移除。请谨慎使用，并注意它们可能无法按预期工作。

**警告：** 实验性功能可能会出现意外行为，包括潜在的数据丢失或安全漏洞。启用这些功能需自行承担风险。

---

## 启用实验性功能

要启用或禁用实验性功能：

1.  打开 Roo Code 设置（右上角的 `<Codicon name="gear" />` 图标）。
2.  进入“高级设置”部分。
3.  找到“实验性功能”部分。

---

## 当前可用的实验性功能

以下实验性功能当前可用：

- [自定义工具](/features/experimental/custom-tools) - 定义 TypeScript/JavaScript 工具，Roo 可以像内置工具一样调用它们
- [并发文件编辑](/features/experimental/concurrent-file-edits) - 在单个操作中编辑多个文件
- [后台编辑](/features/experimental/background-editing) - 在 Roo 后台编辑文件时不受干扰地工作
- [图像生成](/features/image-generation) - 根据文本提示生成图像并保存到工作区
- [运行斜杠命令](/advanced-usage/available-tools/run-slash-command) - 执行预定义的斜杠命令以进行模板化指令和工作流自动化
