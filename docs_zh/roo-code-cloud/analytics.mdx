# 使用情况分析

Roo Code Cloud 提供全面的分析功能，帮助您了解团队如何使用 AI、优化成本并衡量生产力。

<img src="/img/analytics/overview.png" alt="分析仪表板" width="800" />

## 访问方式

前往 Roo Code Cloud 中的[分析](https://app.roocode.com/usage)标签页。

---

## 跟踪内容

- 对于 Cloud Agents，所有任务都会被跟踪并纳入分析
- 对于扩展程序，只有在登录状态且启用了[任务同步](/roo-code-cloud/task-sync)的情况下运行的任务才会被报告用于分析。您可以关闭任务同步，这样任务内容就不会发送到云端，但只要您登录了扩展程序，使用情况数据仍会被报告。这样您就可以将某些任务的内容保留在本地，同时仍能享受使用情况分析带来的好处。
