# 使用分析

Roo Code Cloud 提供全面的分析功能，帮助您了解团队如何使用 AI、优化成本并衡量生产力。

<img src="/img/analytics/overview.png" alt="Analytics Dashboard" width="800" />

## 访问方式

前往 Roo Code Cloud 中的 [Analytics](https://app.roocode.com/usage) 选项卡。

---

## 跟踪内容

- 对于云端代理，所有任务都会被跟踪并计入分析
- 对于扩展程序，只有在登录且启用 [任务同步](/roo-code-cloud/task-sync) 时运行的任务才会报告给分析系统。您可以关闭任务同步，这样任务内容就不会发送到云端，但只要您登录了扩展程序，使用数据始终会被上报。这使得您可以在将某些任务内容保留在本地的同时，仍然受益于使用分析功能。
