# Roo Code Cloud 概述

Roo Code Cloud 通过在云端独立运行的自主 [云端代理](/roo-code-cloud/cloud-agents) 来改变 AI 开发。这些代理不再只是在编辑器中辅助你，而是能够自主地进行规划、编码、审查和修复问题，你可以直接从 **GitHub PR**、**Slack 消息** 或 **网页浏览器** 中访问它们。

它还增强了 Roo Code VS Code 扩展的功能，包括：
- [任务同步](/roo-code-cloud/task-sync)：在一个地方访问你的完整任务历史
- [任务共享](/roo-code-cloud/task-sharing)：与朋友或同事共享特定任务
- [远程控制](/roo-code-cloud/roomote-control)：在远离电脑时通过扩展启动/停止并控制任务
- [Token 使用分析](/roo-code-cloud/analytics)：了解你的 Token 和费用消耗情况

此外，团队版还提供面向团队的功能：
- 团队范围的分析
- 集中式提供商设置和其他配置
- 共享访问同一云端代理团队

## 快速开始

要使用它，你需要一个账户。你可以：
- **[直接注册 Roo Code 账户](https://app.roocode.com/sign-up)** 并设置你的代理团队
- 或者 **[从 VS Code 扩展注册 Roo Code 账户](/roo-code-cloud/login)** 登录你的账户

如果你在注册时选择了 Roo Code Cloud 提供商，那么你已经拥有一个账户，只需登录即可。
