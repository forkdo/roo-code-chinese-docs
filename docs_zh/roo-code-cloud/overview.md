# Roo Code Cloud 概览

Roo Code Cloud 通过自主的[云代理](./cloud-agents)彻底改变了 AI 开发，这些代理在云端独立工作。它们不仅能协助您在编辑器中工作，还能自主规划、编码、审查和修复问题，并可直接通过 **GitHub PR**、**Slack 消息**或**网页浏览器**访问。

它还通过添加以下功能增强了 Roo Code VS Code 扩展的能力：
- 访问 [Roo Code 路由器](/roo-code-router/overview)
- [任务同步](./task-sync)，在一个地方访问您的整个任务历史记录
- [任务共享](./task-sharing)，与朋友或同事共享特定任务
- [远程控制台](./roomote-control)，在远离计算机时启动/停止和控制扩展中的任务
- [令牌使用分析](./analytics)，了解您的令牌和费用的使用情况

此外，在[团队计划](./team-plan)中，还增加了面向团队的功能：
- 集中式 LLM 推理计费，无需管理 API 密钥（通过 [Roo Code 路由器](/roo-code-router/overview)）
- 全团队的任务历史记录和令牌使用分析
- 共享的云代理团队（集中式计费）
- 与 GitHub 和 Slack 的集中式集成（非每个用户单独集成）
- 扩展配置强制执行（提供商/模型、MCP 服务器等）

## 开始使用

要使用它，您需要一个账户。您可以：
- **[直接注册 Roo Code 账户](https://app.roocode.com/sign-up)** 并设置您的代理团队
- 或 **[通过 VS Code 扩展注册 Roo Code 账户](/roo-code-cloud/login)** 到您的账户

如果您在注册时选择了 Roo Code 路由器，那么您已经拥有一个账户，只需登录即可。

## 管理您的账户

您可以通过 [Web 界面](https://app.roocode.com) 并点击右上角的头像菜单来管理所有账户设置（包括取消订阅）：

- **账户切换器：** 在关联到同一电子邮件地址的多个账户之间切换（如果您有多个账户）。
- **计费和信用：** 添加[信用](/credits/overview)、配置自动充值、更改付款方式、升级计划。
- **我的设置：** 更改特定于您的设置，或取消/删除您的个人 Roo Code 云账户（在“打开个人资料”下）。
- **组织设置：** 更改团队的[全局设置](/roo-code-cloud/team-plan#capabilities)。
