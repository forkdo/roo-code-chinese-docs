将 Roo Code 连接到 Roo Code Cloud，将您的编辑器与 Web 应用关联起来，实现任务同步、在线历史记录和协作功能。

## 快速入门

1. 在 Roo Code 中，点击云图标 <Codicon name="cloud" /> 并选择“连接”。
2. 在 app.roocode.com 上选择登录方式（GitHub/Google OAuth 或邮箱）。
3. 您将返回到 VS Code，此时您的账户已连接。
