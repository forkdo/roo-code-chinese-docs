将 Roo Code 连接到 Roo Code Cloud，将你的编辑器与 Web 应用关联，以实现任务同步、云端历史记录和协作功能。

## 快速开始

1. 在 Roo Code 中，点击云图标 <Codicon name="cloud" /> 并选择“连接”。
2. 在 app.roocode.com 上选择登录方式（GitHub/Google OAuth 或 Email）。
3. 你将返回到 VS Code，账户已连接。
