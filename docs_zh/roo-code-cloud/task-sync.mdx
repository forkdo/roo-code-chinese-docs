# 任务同步/监控

Task Sync 将任务活动从本地 VS Code 流式传输到 Roo Code Cloud，让你可以从任何设备监控进度，并稍后访问任务。此功能对所有用户免费。

它会创建一个从本地 VS Code 到 Roo Code Cloud 的实时单向任务消息和状态流，让你可以：

- 在离开办公桌时通过手机监控长时间运行的任务
- 查看近乎实时的更新（每隔几秒轮询一次）：消息、模式更改和状态
- 无需坐在 IDE 前即可检查任务何时完成
- 随时查看云端任务历史记录

## 开始使用

1. [连接你的 IDE](/roo-code-cloud/connect)（或查看 [登录/注册](/roo-code-cloud/login)）
2. 在本地 Roo Code 扩展中打开或启动任务
3. 访问 https://app.roocode.com 主页以近乎实时地监控

- 个人账户：Task Sync 默认启用，可在设置中切换。
- 组织账户：Task Sync 由组织管理员管理，默认对成员关闭，除非在组织级别启用。

## 升级到远程控制

如果你还想远程继续任务（批准操作、响应提示、启动/停止任务），请启用 [Roomote Control](/roo-code-cloud/roomote-control)。

## 限制

- 仅单向监控；不会向 IDE 发送云端命令
- 要实现实时更新，IDE 必须保持打开并连接状态
- 实时更新需要互联网连接

即使 IDE 关闭，你也可以随时查看云端任务历史记录。

## 隐私与安全

- 你的数据绝不会用于训练或广告目的
- Roo Code 团队无法访问你任务的内容，除非你明确授权我们调试任何问题
