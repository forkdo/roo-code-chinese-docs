# 团队计划

团队计划专为团队使用 Roo Code 套件而设计，无论是 Extension、Cloud 还是两者的组合。

它允许：
- 集中式 LLM 推理计费，无需管理 API 密钥（通过 [Roo Code Router](/roo-code-router/overview)）
- 团队范围的任务历史和令牌使用分析
- 共享的 Cloud Agent 团队（具有集中式计费）
- 与 Github 和 Slack 的集中式集成（非每个用户一个）
- Extension 配置强制（提供商/模型、MCP 服务器等）

您可以通过开始 14 天的免费试用访问团队功能。

如果您已经拥有 Roo Code Cloud 账户，可以[在此处升级](https://app.roocode.com/checkout/team)。如果没有，请从我们的[定价页面](https://roocode.com/pricing)注册。

其他团队计划功能可在以下位置找到：
- [使用分析](./analytics)（显示团队范围内的活动和每个用户的过滤器）
- 任务列表（在“主页”和“Agent”页面中，显示团队范围内的任务）

我们不断扩展和调整面向团队的功能，因此如果您需要其他功能，请随时通过 [support@roocode.com](mailto:support@roocode.com) 联系我们。
