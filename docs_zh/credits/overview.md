# Roo Code 积分概述

Roo Code 由几个产品组成：
- **Roo Code VS Code**（扩展）完全免费
- **Roo Code Cloud Provider**（模型提供商）提供免费和付费模型
- **Roo Code Cloud Agents**（自主代理）是付费功能，因为它们会产生基础设施成本

为了使用我们提供商的付费模型和运行 Cloud 代理，你需要积分。

:::info 最新定价
你可以在我们的 [定价页面](https://roocode.com/pricing) 查看我们提供的计划和最新的定价信息
:::
----

## 什么是积分？

积分只是你可以选择预购的预付费金额，用于使用 Roo Code 套件的付费部分。如果你只使用免费功能和免费模型，就不需要购买任何积分。

积分以美元显示，没有转换成难以计算的模糊货币的花招。

## 积分多少钱？

我们 [定价页面](https://roocode.com/pricing) 始终提供我们计划和 Cloud 代理定价的最新信息。

模型定价可以在我们网站的 [提供商页面](https://roocode.com/provider) 找到。

我们不在此处包含价格，只是为了确保它们始终是最新的。

## 积分和月度订阅有什么关系？

Team 计划的月度订阅只是解锁高级功能。它不包含任何积分。免费计划没有任何定期订阅费用，你只为购买的积分付费。

## 如何获得免费积分或免费分钟或免费代币？

当你注册 Roo Code Cloud 账户时，你会获得一定数量的"免费分钟"来试用产品（你会看到一个可爱的 <LucideIcon name="Gift" style={{ width: 14, position: "relative", top: 4}} /> 图标）。这些免费分钟涵盖 Cloud Agent 的运行时间 _和_ Roo Code Cloud 提供商的推理（这两项通常都需要积分）。这些不能在扩展中使用——它们是让你体验 Cloud 代理魔力的。

如果你想在扩展中免费推理，选择一个免费模型（或另一个可能免费为你提供的提供商）。

## 如何购买积分？

### 单次购买

从你的 [Roo Code Cloud 账户](https://app.roocode.com.) 进行一次性购买：

1. 点击右上角的积分余额显示
    <img src="/img/credits/credit-balance-zero.png" alt="积分余额显示" width="180" />

2. 选择"购买积分"
3. 输入你想购买的金额并按照流程操作
4. 你会收到确认邮件，余额会显示在 UI 的同一位置
    <img src="/img/credits/credit-balance.png" alt="积分余额显示" width="180" />

### 自动充值

为了确保你永远不会用完积分，你可以设置自动充值：

1. 点击相同的积分余额显示
    <img src="/img/credits/credit-balance-zero.png" alt="积分余额显示" width="180" />

2. 点击 ⚡ 图标
    <img src="/img/credits/auto-top-up-1.png" alt="积分余额下拉菜单" width="280" />

3. 启用选项并配置阈值和购买金额
4. 点击保存，完成设置

如果你改变主意，只需按照相同的步骤关闭它即可。

你也可以从你的 [账单页面](https://app.roocode.com/billing)（用户头像菜单 → 账单和积分）访问所有这些功能。
