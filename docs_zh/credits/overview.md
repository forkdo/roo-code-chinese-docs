# Roo Code Credits 概览

Roo Code 由以下几部分组成：
- **Roo Code VS Code**（扩展插件）完全免费
- **Roo Code Router**（模型提供商）提供免费和付费模型
- **Roo Code Cloud Agents**（自主代理）是付费功能，因为它们会产生基础设施成本

要使用我们提供商提供的付费模型以及运行 Cloud Agents，您需要购买 Credits。

:::info 最新定价信息
您可以在我们的[定价页面](https://roocode.com/pricing)查看我们提供的套餐以及最新的定价信息
:::
----

## 什么是 Credits？

Credits 只是您可以选择购买的预付费美元金额，用于使用 Roo Code 套件的付费部分。如果您坚持使用免费功能并使用免费模型，则无需购买任何 Credits。

Credits 以美元显示，不会将其转换为难以计算的晦涩货币。

## Credits 的价格是多少？

我们的[定价页面](https://roocode.com/pricing)始终提供我们套餐和 Cloud Agent 定价的最新信息。

模型定价可以在我们网站的[提供商页面](https://roocode.com/provider)找到。

我们在此不包含价格，只是为了确保它们始终保持最新。

## Credits 和月度订阅有什么关系？

Team 计划的月度订阅只是解锁高级功能，它不包含任何 Credits。而免费计划没有任何经常性订阅费用，您只需为您购买的 Credits 付费。

## 如何获得免费 Credits 或免费分钟数或免费令牌？

当您注册 Roo Code Cloud 账户时，您将获得一定数量的"免费分钟数"来试用产品（您会看到一个可爱的 <LucideIcon name="Gift" style={{ width: 14, Position: "relative", top: 4}} /> 图标）。这些免费分钟数涵盖了[Cloud Agent](/roo-code-cloud/cloud-agents) 的运行时间_以及_通过 Roo Code Cloud 提供商进行的推理（这两项通常都需要消耗 Credits）。这些不能在扩展插件中使用——它们是为了让您体验 Cloud Agents 的魔力。

如果您想在扩展插件中进行免费推理，请在可用时寻找免费模型（或使用其他可能免费提供给您使用的提供商）。

## 如何购买 Credits？

### 单次购买

从您的 [Roo Code Cloud 账户](https://app.roocode.com.) 进行一次性购买：

1. 点击右上角的 Credits 余额显示
    <img src="/img/credits/credit-balance-zero.png" alt="Credit balance display" width="180" />

2. 选择"购买 Credits"
3. 输入您要购买的金额并按照流程操作
4. 您将收到确认电子邮件，余额将在 UI 中的同一位置显示
    <img src="/img/credits/credit-balance.png" alt="Credit balance display" width="180" />

### 自动充值

为确保您永远不会用完 Credits，您可以设置自动充值：

1. 点击相同的 Credits 余额显示
    <img src="/img/credits/credit-balance-zero.png" alt="Credit balance display" width="180" />

2. 点击 ⚡ 图标
    <img src="/img/credits/auto-top-up-1.png" alt="Credit balance dropdown" width="280" />

3. 启用该选项并配置阈值和购买金额
4. 点击保存即可完成

如果您改变主意，只需按照相同的步骤操作，同时将其关闭即可。

您也可以从[计费页面](https://app.roocode.com/billing)（用户头像菜单 → 计费和 Credits）访问所有这些功能。
