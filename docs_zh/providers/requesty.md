# 在 Roo Code 中使用 Requesty

Roo Code 支持通过 [Requesty](https://www.requesty.ai/) AI 平台访问模型。Requesty 提供了一个简单且经过优化的 API，用于与 150+ 种大语言模型 (LLM) 进行交互。

**网站:** [https://www.requesty.ai/](https://www.requesty.ai/)

---

## 获取 API 密钥

1.  **注册/登录:** 访问 [Requesty 网站](https://www.requesty.ai/) 并创建账户或登录。
2.  **获取 API 密钥:** 您可以在 Requesty 仪表盘的 [API 管理](https://app.requesty.ai/manage-api) 部分获取 API 密钥。

---

## 可用模型

Roo Code 会自动从 Requesty API 获取所有可用的模型（来自多个提供商的 150+ 种模型）。

如需查看完整、最新的模型列表和定价信息，请访问 [Requesty 的模型列表](https://app.requesty.ai/router/list)。

---

## 在 Roo Code 中的配置

1.  **打开 Roo Code 设置:** 点击 Roo Code 面板中的齿轮图标 (<Codicon name="gear" />)。
2.  **选择提供商:** 在 "API Provider" 下拉菜单中选择 "Requesty"。
3.  **输入 API 密钥:** 将您的 Requesty API 密钥粘贴到 "Requesty API Key" 字段中。
4.  **选择模型:** 从 "Model" 下拉菜单中选择您需要的模型。

---

## 提示和注意事项

- **成本优化**: Requesty 提供多种飞行中的成本优化功能，帮助您降低成本。
- **统一简化的账单**: 通过单一 [API 密钥](https://app.requesty.ai/manage-api) 无限制访问所有提供商和模型，自动余额充值等。
- **成本追踪**: 通过 [成本仪表盘](https://app.requesty.ai/cost-management) 或 [Requesty VS Code 扩展](https://marketplace.visualstudio.com/items?itemName=Requesty.requesty) 追踪每种模型、编程语言、修改文件等的成本。
- **统计数据和日志**: 查看您的 [编码统计仪表盘](https://app.requesty.ai/usage-stats) 或浏览您的 [LLM 交互日志](https://app.requesty.ai/logs)。
- **降级策略**: 当提供商服务中断时，通过降级策略确保您的 LLM 仍能正常工作。
- **提示缓存**: 部分提供商支持提示缓存。[搜索支持缓存的模型](https://app.requesty.ai/router/list)。
