# 在 Roo Code 中使用 Cerebras

Cerebras AI 专注于极快的推理速度（高达 2600 tokens/秒），并提供有竞争力的定价，包括免费套餐。他们的模型针对编码、通用智能和推理任务进行了优化。

**官网：** [https://cloud.cerebras.ai/](https://cloud.cerebras.ai/)

---

## 获取 API 密钥

1. **注册/登录：** 访问 [Cerebras Cloud](https://cloud.cerebras.ai?utm_source=roocode)，创建账户或登录。
2. **进入 API 密钥页面：** 在仪表盘中找到 API 密钥部分。
3. **创建密钥：** 生成一个新的 API 密钥。为其设置一个描述性名称（例如 "Roo Code"）。
4. **复制密钥：** **重要：** 立即复制 API 密钥，并安全保存。

---


## 在 Roo Code 中配置

1. **打开 Roo Code 设置：** 点击 Roo Code 面板中的齿轮图标 (<Codicon name="gear" />)。
2. **选择提供商：** 在“API Provider”下拉菜单中选择 "Cerebras"。
3. **输入 API 密钥：** 将 Cerebras API 密钥粘贴到 "Cerebras API Key" 字段中。
4. **选择模型：** 从 "Model" 下拉菜单中选择所需的模型。

---

## 可用模型

Roo Code 会自动从 Cerebras AI 的 API 获取所有可用模型。

有关完整的最新模型列表和定价信息，请访问 [Cerebras Cloud](https://cloud.cerebras.ai?utm_source=roocode)。
