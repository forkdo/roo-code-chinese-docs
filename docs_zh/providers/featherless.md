# 在 Roo Code 中使用 Featherless AI

Featherless AI 提供对高性能开源模型的访问，包括 DeepSeek、Qwen 和其他大型语言模型。所有模型目前均可免费使用，是预算有限的开发者的理想选择。

**网站：** [https://featherless.ai](https://featherless.ai)

---

## 获取 API 密钥

1. **注册/登录：** 访问 [Featherless AI](https://featherless.ai)，创建账户或登录。
2. **导航到 API 密钥：** 在账户中访问 [API 密钥页面](https://featherless.ai/account/api-keys)。
3. **创建密钥：** 生成一个新的 API 密钥。为其提供一个描述性名称（例如 "Roo Code"）。
4. **复制密钥：** **重要：** 立即复制 API 密钥。它只会显示一次。请安全存储。

---

## 可用模型

Roo Code 会自动从 Featherless AI 的 API 获取所有可用模型。

如需查看完整且最新的模型列表，请访问 [Featherless AI](https://featherless.ai)。

---

## 在 Roo Code 中的配置

1. **打开 Roo Code 设置：** 点击 Roo Code 面板中的齿轮图标 (<Codicon name="gear" />)。
2. **选择提供商：** 从“API 提供商”下拉菜单中选择“Featherless AI”。
3. **输入 API 密钥：** 将你的 Featherless API 密钥粘贴到“Featherless API 密钥”字段中。
4. **选择模型：** 从“模型”下拉菜单中选择你想要的模型。
