# 在 Roo Code 中使用 Featherless AI

Featherless AI 提供对 DeepSeek、Qwen 等高性能开源大语言模型的访问。目前所有模型均可免费使用，是注重预算的开发者的理想选择。

**官网：** [https://featherless.ai](https://featherless.ai)

---

## 获取 API 密钥

1. **注册/登录：** 访问 [Featherless AI](https://featherless.ai)。创建账户或登录。
2. **进入 API 密钥页面：** 在账户中访问 [API 密钥页面](https://featherless.ai/account/api-keys)。
3. **创建密钥：** 生成一个新的 API 密钥。为其指定一个描述性名称（例如，“Roo Code”）。
4. **复制密钥：** **重要：** 立即复制该 API 密钥。它仅会显示一次，请妥善保存。

---

## 可用模型

Roo Code 会自动从 Featherless AI 的 API 获取所有可用模型。

如需查看完整且最新的模型列表，请参见 [Featherless AI](https://featherless.ai)。

---

## 在 Roo Code 中配置

1. **打开 Roo Code 设置：** 点击 Roo Code 面板中的齿轮图标 (<Codicon name="gear" />)。
2. **选择服务提供商：** 在 “API Provider” 下拉菜单中选择 “Featherless AI”。
3. **输入 API 密钥：** 将您的 Featherless API 密钥粘贴到 “Featherless API Key” 字段中。
4. **选择模型：** 从 “Model” 下拉菜单中选择您需要的模型。
