# Roo Code Cloud Provider

为了让使用 Roo Code 变得尽可能顺畅（Roo Code 依赖 LLM 推理服务提供商），我们构建了 Roo Code Cloud Provider。

无需单独订阅或管理 API 密钥。它完全集成在我们的产品套件中，易于上手。功能包括：

- 免费和付费模型
- 仅提供经过测试、与 Roo 兼容性良好的模型
- 付费模型按成本价提供，无加价
- 与您在 Roo Code Cloud Agents 中使用的相同预付费积分机制
