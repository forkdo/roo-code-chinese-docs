# 在 Roo Code 中使用 Chutes AI

Chutes.ai 提供对多种大语言模型（LLM）的免费 API 访问，允许开发者在没有即时财务承诺的情况下集成和实验这些模型。他们提供精选的开源和专有语言模型，通常专注于特定功能或地区语言支持。

**网站：** [https://chutes.ai/](https://chutes.ai/)

---

## 获取 API 密钥

要在 Roo Code 中使用 Chutes AI，请从 [Chutes AI 平台](https://chutes.ai/) 获取 API 密钥。注册或登录后，您应该能够在账户仪表板或设置中找到生成或检索 API 密钥的选项。

---

## 可用模型

Roo Code 会自动从 Chutes AI 的 API 获取所有可用模型。

如需完整、最新的模型列表，请参阅 [Chutes AI 平台](https://chutes.ai/) 或您的账户仪表板。

---

## 在 Roo Code 中的配置

1.  **打开 Roo Code 设置：** 点击 Roo Code 面板中的齿轮图标（<Codicon name="gear" />）。
2.  **选择提供商：** 从“API Provider”下拉菜单中选择“Chutes AI”。
3.  **输入 API 密钥：** 将您的 Chutes AI API 密钥粘贴到“Chutes AI API Key”字段中。
4.  **选择模型：** 从“Model”下拉菜单中选择您需要的模型。
