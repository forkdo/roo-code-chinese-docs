Roo Code 3.23 版本说明

此版本将代码库索引功能提升至稳定功能，引入了强大的新待办清单来管理复杂任务，并包含关键的性能改进和错误修复！

代码库索引：始终开启，随时就绪​

代码库索引已从实验性功能毕业，现已成为 Roo Code 的核心功能，可直接从聊天输入中使用。一旦配置完成，索引器将在后台自动运行，确保 Roo 始终对您的项目保持最新的语义理解。要免费开始使用，请参阅代码库索引快速入门指南。

感谢 MuriloFP、OleynikAleksandr、sxueck、CW-B-W、WAcry、bughaver、daniel-lxs、SannidhyaSah、ChuKhaLi、HahaBill、koberghe、sfz009900、tmchow、rxpjd、buck-0x 和 BenWilles 帮助此功能顺利上线！ (#5378, #5377, #5383, #5483, #5481, #5228, #5214, #5158, #5402, #5451, #5352, #5403, #5785)

新功能：任务待办清单​

此版本引入了一项新的待办清单功能，帮助您跟踪复杂任务。Roo Code 现在会为您的任务显示一个步骤清单，确保不会遗漏任何步骤。您可以直接在聊天界面中查看和管理待办清单。

感谢 qdaxb 提供此功能！ (#5182, #5472, #5473, #5482)

新功能：命令拒绝列表​

我们添加了自动拒绝工作流中不需要的命令的功能：

  • 始终拒绝：将命令标记为“始终拒绝”，以防止意外执行
  • 节省时间：无需重复手动拒绝相同的命令
  • 工作流控制：通过“始终拒绝”选项补充现有的自动批准功能

这为您在自动化工作流中的命令执行提供了更好的控制。

Roo Code Cloud 等候名单​

我们很高兴宣布 Roo Code Cloud 的早期访问注册！（感谢 brunobergher！）

  • 网站注册：网站标题中新增行动号召按钮，方便注册等候名单
  • 扩展内访问：完成 3 个以上任务的用户将直接在扩展中看到等候名单注册选项
  • 早期访问：成为首批体验 Roo Code Cloud 强大功能的用户

立即加入等候名单，抢先体验云端功能！

Claude Code 的 Windows 支持​

我们为 Claude Code 提供程序添加了原生 Windows 支持（感谢 SannidhyaSah、kwk9892！）（#5615）：

  • 修复 Windows 兼容性：Claude Code 提供程序在 Windows 系统上不再卡住
  • 改进输入处理：使用基于 stdin 的输入，消除命令行长度限制
  • 跨平台：在 Windows、macOS 和 Linux 上稳定运行

生活质量改进​

  • 更智能的命令验证：改进的命令验证逻辑，仅在子 shell 命令包含被拒绝的前缀时才自动拒绝，使系统在允许哪些命令方面更加智能
  • 多行命令支持：现在可以正确解析包含换行符的命令，使跨不同操作系统（Unix、Windows 和旧 Mac 格式）使用多行命令更加容易
  • 简化的提示历史记录：当光标位于输入字段开头时，仅使用上下箭头键即可导航提示历史记录 (#5467)
  • API 提供程序搜索：现在您可以在设置中搜索和筛选 API 提供程序列表（感谢 GOODBOY008！）（#5278）
